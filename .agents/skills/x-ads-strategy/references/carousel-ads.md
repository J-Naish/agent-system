# X広告フォーマット：カルーセル広告

---

## フォーマット概要

カルーセル広告は水平スワイプで2〜6枚の画像または動画を表示できるフォーマットであり、複数の商品紹介やストーリーテリングに最適である。1つの広告で複数のメッセージやリンク先を設定でき、ユーザーのスワイプ操作によるインタラクションを促進する。

シングルデスティネーション（全スライド共通のURL）とマルチデスティネーション（各カードに異なるURL）の2つのリンクタイプに対応しており、目的に応じた柔軟な設計が可能である。

### 基本仕様

| 項目 | 仕様 |

| 項目 | 仕様 |
|------|------|

| 対応フォーマット | MP4, MOV |

### リンクタイプ

| タイプ | 説明 |
|--------|------|
| シングルデスティネーション | 全スライドが1つのURLにリンク。ストーリーテリングやブランド訴求向け |
| マルチデスティネーション | 各カードに異なるURL（最大6つ）。商品カタログやEC向け |

## クリエイティブベストプラクティス

### カード配列戦略

カルーセル広告では、カードの配列順序がパフォーマンスに大きく影響する。以下の3つのパターンを目的に応じて使い分ける。

#### パターン1：ストーリーテリング型

```
カード1: 問題提起（ターゲットの課題を提示）
  ↓ スワイプ
カード2: 解決策（自社サービス/商品の紹介）
  ↓ スワイプ
カード3: 事例/証拠（導入事例、データ、Before/After）
  ↓ スワイプ
カード4: CTA（行動喚起、オファー提示）
```

#### パターン2：プロダクトカタログ型

```
カード1: ベストセラー商品（最も人気のある商品を最初に）
  ↓ スワイプ
カード2: 関連商品A
  ↓ スワイプ
カード3: 関連商品B
  ↓ スワイプ
カード4: まとめCTA（全商品を見る、カテゴリページへ）
```

#### パターン3：機能紹介型

```
カード1: ヘッドライン + 全体概要
  ↓ スワイプ
カード2: 機能A（具体的なベネフィット）
  ↓ スワイプ
カード3: 機能B（具体的なベネフィット）
  ↓ スワイプ
カード4: CTA（無料トライアル、デモ申込等）
```

### 最適カード枚数

- **3〜5枚が最適**：2枚では情報が少なすぎ、6枚は最後まで見られることが稀
- **最も重要なカードを1枚目に配置**：スワイプ率はカードが進むほど低下する
- **最後のカードは必ずCTAカード**：スワイプしたユーザーに明確な行動を促す

### デザインルール

| ルール | 詳細 |
|--------|------|
| 統一アスペクト比 | 全カードで同じアスペクト比を使用（混在は不可） |
| 一貫したデザイン | カード間でカラー、フォント、スタイルを統一する |
| 各カード単独で意味が通じる | 全カードを見なくても、1枚だけで内容が伝わる設計 |
| スワイプの視覚的キュー | 矢印や画像の切れ目で「スワイプできる」ことを暗示する |

### よくある間違い

| 間違い | 問題点 |
|--------|--------|
| アスペクト比の混在 | 仕様上不可。統一が必須 |
| カード間でデザインがバラバラ | 統一感がなく、プロフェッショナルな印象が損なわれる |
| 1枚目にインパクトがない | スワイプされず、残りのカードが見られない |
| CTAカードがない | スワイプしたユーザーの行動先が不明確 |
| 6枚全てに情報を詰め込む | 後半カードは見られない前提で設計すべき |

## 広告コピー構成

### 投稿テキスト

- **50〜100文字でカルーセル全体のテーマを伝える**：個別カードの説明ではなく、全体の文脈を設定する
- 投稿テキストは常に表示されるため、カルーセルを見る動機を与える役割

### カード別ヘッドライン

- **各カード50文字以内**：カード上に表示されるヘッドラインは簡潔に
- **マルチデスティネーションの場合、各ヘッドラインはリンク先LPの内容と一致させる**：クリック後の期待値とLPの内容にズレがあると離脱率が上昇する

### コピー構成例

```
【投稿テキスト】（全体テーマ：50〜100文字）
  ↓
【カード1ヘッドライン】（50文字以内）
【カード2ヘッドライン】（50文字以内）
【カード3ヘッドライン】（50文字以内）
【カード4ヘッドライン】（50文字以内）= CTAカード
```

## ファネル/フロー設計

### シングルデスティネーション：ストーリーテリング/ブランド訴求

```
カルーセル広告（ストーリーテリング型配列）
  ↓
全カードが同一LPにリンク
  ↓
LP上でカルーセルの流れを引き継ぐストーリー展開
  ↓
コンバージョン
```

- カルーセルで問題提起→解決策→事例→CTAの流れを作り、LPでその続きを展開
- LP上の構成がカルーセルの流れと一貫していることが重要
- ブランドストーリーや新商品ローンチに最適

### マルチデスティネーション：EC/商品カタログ

```
カルーセル広告（プロダクトカタログ型配列）
  ↓
各カードが個別の商品詳細ページにリンク
  ↓
商品A → 商品A詳細ページ
商品B → 商品B詳細ページ
商品C → 商品C詳細ページ
  ↓
各商品ページでコンバージョン
```

- 各カードのビジュアル・ヘッドラインとリンク先の商品詳細ページが完全に一致すること
- カード上の商品画像 → クリック → 同じ商品の詳細ページという一貫性が必須
- ECサイト、複数サービスの紹介、不動産物件一覧などに最適

### ファネル段階別の活用

| ファネル段階 | 推奨パターン | リンクタイプ |
|-------------|-------------|-------------|
| TOFU（認知） | ストーリーテリング型 | シングルデスティネーション |
| MOFU（検討） | 機能紹介型 | シングルデスティネーション |
| BOFU（行動） | プロダクトカタログ型 | マルチデスティネーション |

### メッセージ一貫性の原則

- **シングルデスティネーション**：カルーセル全体のストーリー → LP上で同じ流れを継続
- **マルチデスティネーション**：各カードのヘッドライン・ビジュアル → 各リンク先ページの内容が完全一致

---

## まとめ

カルーセル広告は複数の商品やメッセージを1つの広告で伝えられる強力なフォーマットであり、アプリインストール+24%、CTR+15%、サイト訪問コンバージョン+25%という優れた実績を持つ。成功の鍵は、カード配列戦略の適切な選択、1枚目のインパクト最大化、最後のCTAカード設置、そして全カードを通じたデザインの統一性である。シングル/マルチデスティネーションの使い分けにより、ブランディングからEC販売まで幅広い目的に対応できる。

# Search Keyword Ads（検索キーワード広告）

## フォーマット概要

ユーザーが特定のキーワードやハッシュタグを検索した際に、検索結果の最上部に表示される広告フォーマット。購買意欲の高いユーザーにリーチできる、X上でのインテントマーケティング手法。2025年に新設された専用の広告目的「キーワード」を使用する。

Googleの検索広告と同様に、ユーザーの検索意図に対して広告を表示するため、X広告の中で最も購買意欲の高いユーザーにアプローチできるフォーマット。

### 配置面・目的・課金

- **配置面**: 検索結果の最上部
- **広告目的**: キーワード（2025年新設の専用目的）
- **課金モデル**: CPM（インプレッション課金）

## クリエイティブ仕様

### テキスト仕様

| 項目 | 仕様 |
|------|------|
| 広告テキスト | 最大280文字（リンク含む場合はリンクで23文字消費） |
| 対応フォーマット | テキスト / 画像付き / 動画付き |

### キーワード仕様

| 項目 | 仕様 |
|------|------|
| 最小キーワード数 | 25キーワード / キャンペーン |
| 最大キーワード数 | 750キーワード / 広告グループ |
| マッチング | スペルミス・表記ゆれを含む自動マッチング |

---

## ベストプラクティス

### クリエイティブの基本方針

検索キーワード広告のクリエイティブは、ユーザーの検索意図に対する**「最良の回答」**として設計する。ユーザーは能動的に情報を探しているため、その検索に対して最も的確な答えを提示する広告が最も高いパフォーマンスを発揮する。

### クリエイティブ最適化

1. **検索意図との一致**: 設定キーワードとクリエイティブの内容を完全に一致させる
2. **テキスト/画像/動画の使い分け**: 商材や検索意図に応じて最適なフォーマットを選択する
3. **簡潔さ**: 検索結果画面での視認性を重視し、情報を詰め込みすぎない
4. **即時的な価値提示**: ユーザーの疑問や課題に対する解決策を冒頭で明示する

### キーワード運用

1. キャンペーンあたり**最低25キーワード**を設定する（必須要件）
2. 広告グループあたり**最大750キーワード**まで設定可能
3. **週次でパフォーマンスをレビュー**し、低パフォーマンスのキーワードを除外する
4. マッチング機能がスペルミスや表記ゆれを自動カバーするため、完全一致の網羅は不要
5. キーワードカテゴリ（後述）ごとに広告グループを分けて管理する

## 広告コピー構成

### コピーの基本方針

検索キーワードに**直接関連した内容**で、ユーザーの質問に答える・課題を解決するコピーを作成する。280文字フルに使う必要はなく、**50〜100文字で的確に伝える**ことを優先する。

### コピー作成のポイント

1. **検索キーワードとの直接的な関連性**を最優先にする
2. ユーザーの質問に答える、または課題を解決する内容にする
3. **50〜100文字**で的確・簡潔に伝える（280文字を使い切る必要はない）
4. 検索意図に合った具体的なベネフィットを提示する
5. 広告グループごとにキーワードカテゴリに合わせたコピーを用意する

### キーワードカテゴリ別コピー例

#### ブロードキーワード（認知向け）

業界用語や一般的なカテゴリワードで幅広くリーチする。

| キーワード例 | コピー例 |
|-------------|---------|
| スキンケア | 「肌悩みに合わせた本格スキンケア。まずは無料診断から」 |
| プロジェクト管理 | 「チームの生産性を変えるプロジェクト管理ツール」 |
| 英会話 | 「続けられる英会話。1日15分から始められます」 |

#### スペシフィックキーワード（コンバージョン向け）

商品名・機能名・具体的な課題名でコンバージョンを狙う。

| キーワード例 | コピー例 |
|-------------|---------|
| ニキビ 治し方 | 「繰り返すニキビに。皮膚科医監修の薬用ケア」 |
| タスク管理 ツール 比較 | 「無料で使えるタスク管理。チーム導入実績5,000社」 |
| TOEIC 800点 | 「TOEIC 800点突破者の学習法を公開中」 |

#### コンペティティブキーワード（競合奪取向け）

競合ブランド名の検索を獲得する。

| キーワード例 | コピー例 |
|-------------|---------|
| [競合ブランド名] | 「乗り換えで初月無料。○○との違いを比較」 |
| [競合サービス名] 代替 | 「もっとシンプルに。○○からの移行もかんたん」 |

#### トレンドキーワード（リーチ拡大向け）

話題のトピックやトレンドに乗じてリーチを拡大する。

| キーワード例 | コピー例 |
|-------------|---------|
| [トレンドイベント名] | 「○○シーズン到来。今年のおすすめアイテムはこちら」 |
| [話題のトピック] | 「話題の○○、もう試した？今なら送料無料」 |

## ファネル・導線設計

### 基本ファネル

```
ユーザーがX上でキーワード/ハッシュタグを検索（高い購買意図）
  │
  ▼
検索結果の最上部に広告を表示
  │
  ▼
ユーザーがクリック
  │
  ▼
検索キーワードと一致するメッセージのLP（メッセージマッチ）
  │
  ▼
コンバージョン（X Pixel / CAPI で計測）
```

### ファネル設計のポイント

1. **メッセージマッチの徹底**: 検索キーワード → 広告コピー → LP見出し の一貫性を保つ。ユーザーが検索した内容と広告・LPのメッセージが一致していることが最重要
2. **X上のGoogle検索広告**: ユーザーの検索意図（インテント）に基づくマーケティングという点で、Google検索広告と同じ考え方をX上で実現するフォーマット
3. **最も購買意欲の高いユーザー**: 能動的に検索しているユーザーは、タイムラインで受動的に広告を見るユーザーよりも購買意欲が高い
4. **X Pixel / CAPI 必須**: コンバージョン最適化のためにX PixelまたはCAPIの実装が必須

### キーワードカテゴリ別ファネル戦略

| カテゴリ | 目的 | ファネル上の位置 | LPの種類 |
|---------|------|-----------------|---------|
| ブロードキーワード | 認知拡大 | ファネル上部（TOFU） | ブランド紹介・カテゴリLP |
| スペシフィックキーワード | コンバージョン | ファネル下部（BOFU） | 商品詳細・比較LP |
| コンペティティブキーワード | 競合からの奪取 | ファネル中部〜下部 | 比較・乗り換えLP |
| トレンドキーワード | リーチ拡大 | ファネル上部（TOFU） | トレンド特集LP |

### 計測・最適化

- X Pixel / CAPI でコンバージョンを計測し、キーワードごとのROASを把握する
- 週次でキーワードパフォーマンスをレビューし、低パフォーマンスキーワードを除外する
- キーワードカテゴリ別にCPA・CVRを比較し、予算配分を最適化する
- 検索ボリュームの季節変動を把握し、入札戦略を調整する

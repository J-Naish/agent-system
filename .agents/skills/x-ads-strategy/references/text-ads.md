# X広告フォーマット：テキスト広告

---

## フォーマット概要

テキスト広告はX広告の最もシンプルなフォーマットであり、画像や動画などのメディアを一切使用しない、テキストのみで構成される広告形式である。投稿には「プロモーション」ラベルが表示され、通常のオーガニック投稿と視覚的に区別される。

メディアがないため制作コストが最も低く、コピーの力だけで成果を出す必要がある。テキストのみという制約は、逆にコピーライティングのA/Bテストや会話型ブランディングに最適な環境を提供する。

---

## クリエイティブ仕様

### 文字数制限

| 項目 | 仕様 |
|------|------|
| 英語テキスト | 最大280文字 |
| 日本語テキスト | 最大140文字 |
| リンク | 23文字としてカウント（URL長に関係なく固定） |

### 配信面

| 配信面 | 対応 |
|--------|------|
| タイムライン | ○ |
| 検索結果 | ○ |
| プロフィール | ○ |

### 対応キャンペーン目的

| 目的 | 対応 |
|------|------|
| リーチ | ○ |
| エンゲージメント | ○ |
| ウェブサイトトラフィック | ○ |
| キーワード | ○ |

### 主な使用ケース

- 新機能・新サービス・キャンペーンの告知
- 広告コピーのA/Bテスト（勝ちコピーを画像/動画広告に展開）
- 低予算での広告運用開始
- 会話型ブランディング（ユーザーとの対話促進）

---

## クリエイティブベストプラクティス

### テキスト最適化

- **最適文字数は50〜100文字**：短すぎると情報不足、長すぎるとタイムラインで埋もれる
- **1広告1メッセージ**：複数の訴求を詰め込まない。1つの明確なメッセージに絞る
- **オーガニック投稿のようなトーン**：広告感を出さず、自然な投稿として読まれる文体を心がける
- 「プロモーション」ラベルが付くため、内容自体は広告臭を消すことでCTRが向上する

### 避けるべきこと

- 複数のCTAを1つの投稿に入れる
- 過度なハッシュタグの使用
- 宣伝文句の羅列（「業界No.1」「最安値」等の連発）
- 長文による情報過多

## 広告コピー構成

### コピーパターン

テキスト広告では以下の4つのコピーパターンが効果的である。

#### 1. Question型（質問型）

ユーザーに問いかけることで、思考を促し、エンゲージメントを獲得する。

```
構成：[問いかけ] → [短い補足/回答の示唆]
```

- ターゲットの課題や関心に直接問いかける
- 回答したくなる・リプライしたくなる質問設計
- Yes/Noで終わらないオープンな問いが理想

#### 2. Data型（データ提示型）

具体的な数字やデータを提示し、信頼性と説得力を高める。

```
構成：[インパクトのある数字] → [その意味/示唆] → [行動喚起]
```

- 業界データ、自社実績、調査結果などを活用
- 数字は丸めすぎず、具体的な方がリアリティが出る
- 「〜%のマーケターが」「〜社が導入」など

#### 3. Benefit直球型

ユーザーが得られるベネフィットをストレートに伝える。

```
構成：[ベネフィット（1行目で完結）] → [補足/条件]
```

- 機能ではなくベネフィット（顧客にとっての価値）を訴求
- 1行目で最大の価値を伝えきる
- 補足は最小限に

#### 4. Social Proof型（社会的証明型）

他者の行動や評価を示すことで、信頼と行動を促す。

```
構成：[第三者の評価/実績] → [具体的な成果/数字]
```

- ユーザーレビュー、導入実績、メディア掲載などを活用
- 具体的な名前や数字を入れることで信憑性が増す
- 「〜が選んだ」「〜で紹介された」など

## ファネル/フロー設計

テキスト広告はファネルの入口として、またはコピーテストツールとして2つの主要なフロー設計が可能である。

### フロー1：エンゲージメント起点のリターゲティングフロー

```
テキスト広告（リンクなし）
  ↓
エンゲージメント獲得（いいね、リプライ、RT）
  ↓
エンゲージしたユーザーをオーディエンスとして蓄積
  ↓
画像広告/動画広告でリターゲティング
  ↓
LP → コンバージョン
```

- テキスト広告でリンクを入れず、純粋なエンゲージメントを狙う
- エンゲージしたユーザー＝関心の高いユーザーとして、ビジュアル広告でリターゲティング
- 低コストで質の高いオーディエンスを構築できる

### フロー2：コピーテスト → 勝ちコピー展開フロー

```
テキスト広告で複数コピーパターンをA/Bテスト
  ↓
エンゲージメント率・CTRで勝ちコピーを特定
  ↓
勝ちコピーを画像広告・動画広告のコピーに採用
  ↓
本格的なクリエイティブ広告として展開
```

- テキスト広告は制作コストが低いため、大量のコピーバリエーションをテスト可能
- 勝ちコピーをそのまま画像/動画広告のテキスト部分に転用
- データに基づいたコピー選定により、クリエイティブ広告の成果を最大化

### ファネル上の位置づけ

| ファネル段階 | テキスト広告の役割 |
|-------------|-------------------|
| TOFU（認知） | 会話型ブランディング、話題作り |
| MOFU（検討） | コピーテスト、エンゲージメント蓄積 |
| BOFU（行動） | 直接的なCTA付きテキスト（リンク込み） |

---

## まとめ

テキスト広告は最もシンプルだが、コピーライティングの質が成果を直接左右するフォーマットである。低コストで大量のA/Bテストが可能な点、エンゲージメント起点でリターゲティングオーディエンスを構築できる点が最大の強みである。画像/動画広告の「前段」として活用することで、広告全体の効率を大幅に向上させることができる。

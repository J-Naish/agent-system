# X広告フォーマット：動画広告

---

## フォーマット概要

動画広告はフィード上で自動再生（ミュート状態）され、60秒でオートループする。テキストや画像では伝えきれない動的なメッセージ、デモンストレーション、ストーリーテリングに最適なフォーマットである。

フィードでは音声がミュートで再生されるため、字幕（キャプション）の追加が必須であり、左上にロゴを常時表示することが推奨される。全9種類のキャンペーン目的に対応しており、X広告の中で最も汎用性の高いフォーマットである。

## クリエイティブベストプラクティス

### フック設計（最初の3秒）

動画広告の成否は最初の3秒で決まる。スクロールを止めるフック設計が最重要要素である。

#### 効果的なフック手法

| 手法 | 説明 |
|------|------|
| モーション開始 | 冒頭からアニメーション・動きのある映像で始める |
| 問題提起/衝撃的なビジュアル | ターゲットの課題を視覚的に表現する |
| 人の顔/アクション | 人物が何かをしている映像で注目を集める |
| テキストオーバーレイフック | 大きな文字で1行のフックメッセージを表示 |

#### 避けるべきフック

| NG | 理由 |
|----|------|
| ゆっくりとしたロゴのフェードイン | 3秒をロゴだけで消費し、メッセージが伝わらない |
| 長文テキストの導入 | 読む前にスクロールされる |
| 音なし・動きなし | ミュート再生で静止画と区別がつかない |
| ロゴだけが数秒表示 | ユーザーにとって価値がなく即スキップ |

### 動画構成

```
0〜3秒：フック（Hook）
  スクロールを止める。動き、問題提起、衝撃的なビジュアル
  ↓
3〜10秒：メッセージ（Message）
  ベネフィット提示、デモ、Before/After
  ↓
10〜15秒：CTA（Call to Action）
  行動を促す。エンドカードまたはウェブサイトカード
```

### ファネル段階別の最適尺

| ファネル段階 | 推奨尺 | 目的 |
|-------------|--------|------|
| 認知（Awareness） | 6〜10秒 | ブランド想起、広いリーチ |
| 検討（Consideration） | 15〜30秒 | 詳細説明、デモ、ベネフィット訴求 |
| 行動（Conversion） | 30〜60秒 | 詳細情報、比較、購入動機の形成 |

15秒以下の動画は完了視聴率が+22%向上する。

### サウンド設計

| 要素 | 推奨レベル |
|------|-----------|
| BGM | -15〜-20dB |
| ダイアログ（ナレーション） | -6〜-12dB |
| 効果音 | 控えめに使用 |

- **フィード再生はデフォルトでミュート** → 字幕（キャプション）は必須
- 縦型動画はデフォルトで音声ONになる傾向がある
- BGMはメッセージの邪魔にならないレベルに設定

### ブランディングルール

- **ロゴは常に左上に表示**：動画全体を通じてブランド認知を確保
- **最初の3秒以内にブランドカラーを表示**：視覚的にブランドと紐づける
- **最初の5秒以内に商品を表示**：何の広告かを早期に伝える
- **エンドカード**：ロゴ + CTA + URLを含むエンドカードで動画を締める

### 黄金公式

```
15秒 + 字幕 + 3秒フック + ブランド常時表示 = エンゲージメント+33%
```

## 広告コピー構成

### コピーの役割

動画広告のコピーは動画内容の要約ではなく、**「この動画を見る理由」**を提示する役割を担う。

### コピー設計原則

- **動画の内容をネタバレしない**：コピーで内容を全て説明してしまうと、動画を見る動機が失われる
- **「なぜ見るべきか」を1行で伝える**：動画を再生するモチベーションを与える
- **補足情報を追加**：価格、期間限定、条件など、動画内で伝えきれない情報をコピーで補完
- **最適文字数は50〜100文字**

### コピー構成

```
[なぜこの動画を見るべきか（1行）]
[価値/オファーの補足（1行）]
  ↓
ウェブサイトカードで遷移
```

### 具体的な設計ポイント

| ポイント | 説明 |
|---------|------|
| 1行目 | 動画を見る理由・期待値を設定 |
| 2行目 | 価格、期間、条件など補足情報 |
| CTA | ウェブサイトカードボタンに委ねる |

## ファネル/フロー設計

### 動画広告の3段階ファネル

動画広告は視聴データを活用したリターゲティングが可能であり、ファネル全体を一貫して設計することで最大の効果を発揮する。

#### TOFU（認知 - Top of Funnel）

```
短尺動画（6〜15秒）
  ↓
広いリーチで認知を獲得
  ↓
2秒以上視聴したユーザーをリターゲティングオーディエンスとして蓄積
```

- 目的：ブランド認知、リーチ最大化
- 動画尺：6〜15秒
- KPI：インプレッション数、動画再生数、2秒以上視聴率
- リターゲティング条件：2秒以上視聴したユーザー

#### MOFU（検討 - Middle of Funnel）

```
中尺動画（15〜30秒）
  ↓
TOFU視聴者にリターゲティング配信
  ↓
詳細なベネフィット、デモ、事例を提示
  ↓
エンゲージしたユーザーをさらに蓄積
```

- 目的：検討促進、ベネフィット理解
- 動画尺：15〜30秒
- ターゲット：TOFU動画を2秒以上視聴したユーザー
- KPI：完了視聴率、エンゲージメント率

#### BOFU（行動 - Bottom of Funnel）

```
動画 + ウェブサイトカード
  ↓
MOFUエンゲージャーにリターゲティング配信
  ↓
具体的なオファー、期間限定、CTA
  ↓
LP → コンバージョン
```

- 目的：コンバージョン獲得
- 動画尺：30〜60秒
- ターゲット：MOFU動画にエンゲージしたユーザー
- KPI：CTR、CVR、CPA

### ファネル全体フロー図

```
TOFU: 短尺動画（6-15秒）→ 認知獲得
  ↓ 2秒以上視聴者をリターゲット
MOFU: 中尺動画（15-30秒）→ 詳細訴求
  ↓ エンゲージャーをリターゲット
BOFU: 動画 + ウェブカード → LP → コンバージョン
```

## まとめ

動画広告はX広告で最も汎用性が高く、ファネル全体をカバーできるフォーマットである。成功の鍵は「最初の3秒のフック設計」「15秒以内の尺」「字幕必須」「ブランド常時表示」の4原則を守ること。コピーは動画の要約ではなく「見る理由」を提示し、視聴データを活用したリターゲティングファネルを構築することで、認知から行動までを一気通貫で設計できる。

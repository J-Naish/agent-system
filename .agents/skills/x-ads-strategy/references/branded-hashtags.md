# Branded Hashtags（ブランドハッシュタグ）

## フォーマット概要

Branded Hashtagsは、ブランド専用のハッシュタグにカスタム絵文字/アイコンを紐付け、プラットフォーム上でそのハッシュタグが使用されるたびに自動的にブランド絵文字が表示される広告フォーマット。オーガニックな拡散によりブランド認知を飛躍的に高める。

### 特徴

- ブランド専用ハッシュタグにカスタム絵文字/アイコンが自動表示される
- ハッシュタグを使用した全ての投稿（オーガニック含む）で絵文字が表示される
- ユーザーが自発的にハッシュタグを使うことでオーガニックなブランド露出が拡大する

### 配信面

- ハッシュタグが使用される全てのプラットフォーム上の面（タイムライン、検索結果、トレンド、プロフィール等）

### 価格帯

- プレミアム / マネージドサービス
- 大型イベント向け: $500K〜$1M

## スペック・入稿規定

### 絵文字デザイン仕様

| 項目 | 仕様 |
|------|------|
| サイズ | 72 x 72 px |
| 最小視認サイズ | 16 x 16 px で判読可能であること |
| フォーマット | PNG（透過推奨） |

### ハッシュタグ要件

| 項目 | 条件 |
|------|------|
| 過去30日間のメンション数 | 250回未満 |
| または | 過去30日間のメンションのうち70%以上がブランド関連 |

### 注意事項

- ハッシュタグが既に一般的な用語として広く使われている場合は承認されない
- 複数のハッシュタグバリエーションを申請可能（審査あり）

## ベストプラクティス

### 絵文字デザイン

- **16x16pxでの視認性を最優先**: 実際の表示サイズは極めて小さいため、細かいディテールよりもシルエットの明確さを重視する
- **ブランドアイデンティティの明確な反映**: ロゴの簡略化、ブランドカラーの使用、シンボルマークの採用など、一目でブランドを想起できるデザインにする
- **ライト/ダーク両背景でのテスト**: X はダークモードユーザーが多いため、両方の背景色で視認性を確認する
- **シンプルな形状**: 複雑な形状は縮小時に潰れるため、2〜3色以内、明確なアウトラインのデザインを推奨

### ハッシュタグ設計

- **短く、入力しやすいハッシュタグ**: ユーザーが手入力する可能性があるため、長すぎるハッシュタグは避ける
- **ブランドとの関連性が明確**: ハッシュタグ単体でブランドやキャンペーンが想起できるものにする
- **記憶に残る / シェアしたくなる**: ユニークで語感が良く、会話の中で自然に使えるハッシュタグにする
- **既存の一般用語との重複を避ける**: 審査で却下されるリスクがあるため、ブランド固有の造語やキャンペーン名を推奨

### 統合キャンペーンでの活用

- Branded Hashtags単体ではなく、Trend TakeoverやTimeline Takeoverとの組み合わせで効果を最大化する
- プロモーテッド投稿でハッシュタグの使用を促進し、オーガニック拡散の起点を作る
- インフルエンサーやパートナーにもハッシュタグ使用を依頼し、初期のクリティカルマスを確保する

## コピー構成

### ハッシュタグ自体の設計

```
ハッシュタグ設計のチェックリスト:
- 短い（理想は15文字以内）
- 入力しやすい（複雑なスペルを避ける）
- ブランドとの関連が明確
- 記憶に残る語感
- 一般用語と重複しない
- キャンペーンの文脈で自然に使える
```

### ハッシュタグ促進投稿

```
[キャンペーンメッセージ]

#ブランドハッシュタグ をつけて投稿しよう

[参加メリット / キャンペーン内容]

例:
「あなたの○○体験を教えてください。
#ブランド名チャレンジ をつけて投稿すると、
抽選で○名様に○○をプレゼント」
```

**ポイント**:
- ハッシュタグがキャンペーンのアンカー（核）となるよう設計する
- ユーザーがハッシュタグを使う動機（参加メリット）を明確にする
- ハッシュタグ + カスタム絵文字の視覚的なインパクトを活用して拡散を促す

### UGC促進コピー

```
[ユーザー参加を促す問いかけ / チャレンジ]

#ブランドハッシュタグ で投稿してね

例:
「この春、最初に行きたい場所は？
#ブランド名Spring で教えてください」
```

## ファネル・フロー設計

### 基本フロー

```
Branded Hashtag + カスタム絵文字の設定
    ↓
プロモーテッド投稿でハッシュタグ使用を促進
    ↓
ユーザーがハッシュタグ付きで投稿（オーガニック拡散）
    ↓
カスタム絵文字による視覚的ブランド露出の増幅
    ↓
ブランド認知・好意度の向上
```

### 統合キャンペーンフロー

```
[Phase 1: 認知爆発]
Trend Takeover + Branded Hashtag
    ↓ トレンド枠でハッシュタグの存在を大量露出

[Phase 2: リーチ拡大]
Timeline Takeover + ハッシュタグ促進投稿
    ↓ タイムライン1st viewでハッシュタグ使用を促進

[Phase 3: オーガニック拡散]
ユーザーがハッシュタグ付きで投稿 → カスタム絵文字が自動表示
    ↓ UGCによるオーガニックなブランド露出拡大

[Phase 4: エンゲージメント深化]
ブランドアカウントがUGCにリプライ / リポスト
    ↓ コミュニティ形成・ブランドロイヤルティ醸成

[Phase 5: コンバージョン]
ハッシュタグ参加者をリターゲティング → 購入/申込広告
```

### 推奨する組み合わせパターン

| 組み合わせ | 効果 | 適したシーン |
|-----------|------|-------------|
| Branded Hashtag + Trend Takeover | 最大級の認知インパクト | 大型製品ローンチ、記念日キャンペーン |
| Branded Hashtag + Timeline Takeover | 広範なリーチ + 視覚的ブランディング | シーズナルキャンペーン、ブランドリニューアル |
| Branded Hashtag + Trend Takeover + Timeline Takeover | フルファネル最大化 | 年間最重要キャンペーン、グローバルイベント |
| Branded Hashtag + Branded Notifications | オプトインユーザーからの拡散起点 | コミュニティ主導型キャンペーン |

### 運用上の注意点

- Branded Hashtagsは単体施策ではなく、統合キャンペーンの一部として最大効果を発揮する
- ハッシュタグの審査・承認には時間がかかるため、キャンペーン開始の十分前に申請する
- 絵文字デザインは必ず16x16pxでの表示テストを行い、視認性を確認する
- ライトモード / ダークモード両方での見え方を検証する
- オーガニック拡散を促すために、初動でのプロモーテッド投稿への十分な予算配分が重要
- UGCのモニタリング体制を整え、ネガティブな使用への対応方針を事前に決定しておく
- キャンペーン期間中のハッシュタグ使用数、絵文字表示回数、関連会話量を日次で追跡する

# Collection Ads（コレクション広告）

## フォーマット概要

ショッピング特化型の広告フォーマット。ヒーロー画像（メインビジュアル）と複数のサムネイル画像を組み合わせ、全商品を一画面で一覧表示する。カルーセル広告がスワイプ操作を必要とするのに対し、コレクション広告はスワイプ不要で全商品を同時に見せられる点が最大の差別化ポイント。

### カルーセル広告との違い

| 項目 | コレクション広告 | カルーセル広告 |
|------|-----------------|---------------|
| 商品表示 | 全商品を一画面で一覧表示 | スワイプで1枚ずつ表示 |
| ユーザー操作 | スワイプ不要 | スワイプ必要 |
| 適性 | 複数商品の一括訴求、EC商品一覧 | ストーリーテリング、段階的訴求 |

### 配置面・目的

- **配置面**: タイムライン
- **広告目的**: ウェブサイトのトラフィック、売上

### パフォーマンス実績

- CTR（クリック率）: **+42%**（他フォーマット比）
- インプレッションあたりコンバージョン: **+54%**（他フォーマット比）

## クリエイティブ仕様

### 画像仕様

| 項目 | 仕様 |
|------|------|
| 画像枚数 | 2〜6枚（5枚以上推奨 ※X公式推奨） |
| 画像サイズ | 800 x 800 px |
| アスペクト比 | 1:1 |
| ファイル形式 | PNG / TIFF |
| リンク | 最大6つの異なるリンクを設定可能 |
| 表示形式 | グリッド表示 |

### ヒーロー画像（メインビジュアル）

- **ライフスタイルショットが効果的**: 商品を使用しているシーン（着用写真、使用風景など）
- **解像度**: 800 x 800 の高解像度画像
- **役割**: ブランドの世界観を伝える「入口」として機能する
- 商品単体の切り抜き画像よりも、使用シーンを見せる画像の方がエンゲージメントが高い

### サムネイル画像

- **枚数**: 5枚以上推奨（X公式推奨値）
- **背景**: 白背景の商品画像がEC向けでは最もCTRが高い
- **価格帯**: 複数の価格帯の商品をバリエーション豊かに配置する
- **リンク**: 各サムネイルごとに個別の商品ページへリンクを設定する

---

## ベストプラクティス

### ヒーロー画像の最適化

1. ブランドの世界観が伝わるライフスタイル画像を使用する
2. 高解像度（800x800以上）で鮮明な画像を用意する
3. テキストオーバーレイは最小限にし、ビジュアルで語る
4. 季節感やトレンドを反映した画像を定期的に更新する

### サムネイル構成の最適化

1. 5枚以上のサムネイルでグリッドを埋め、視覚的なインパクトを最大化する
2. EC商品は白背景の切り抜き画像を使用し、商品を明確に見せる
3. 価格帯の異なる商品を混在させ、幅広い層にアプローチする
4. 各サムネイルに個別の商品ページURLを設定し、直接購入導線を作る
5. 売れ筋商品やセール商品を優先的に配置する

### 運用上の注意点

- 商品画像の統一感を保ちつつ、バリエーションを持たせる
- 在庫切れ商品のサムネイルは速やかに差し替える
- A/Bテストでヒーロー画像の効果を検証する（ライフスタイル vs 商品フォーカスなど）

## 広告コピー構成

### コピーの基本方針

コレクション広告のコピーは**短文でコレクションのテーマを伝える**ことに徹する。個々の商品の詳細説明はサムネイルのリンク先LP（商品ページ）に委ねる。

### 効果的なフレーミング例

| フレーミング | コピー例 |
|-------------|---------|
| 新作訴求 | 「新作コレクション」「NEW ARRIVAL」 |
| キュレーション | 「今週のおすすめ」「スタッフが選んだTOP PICKS」 |
| セール | 「SPRING SALE」「期間限定セレクション」 |
| シーズン | 「春の新生活アイテム」「夏のマストハブ」 |
| ランキング | 「売れ筋TOP5」「今月の人気アイテム」 |

### コピー作成のポイント

1. **短く、テーマが一目で伝わる**コピーにする
2. 商品スペックや価格の詳細はコピーに含めない（サムネイルLPに任せる）
3. コレクション全体の魅力や世界観をワンフレーズで表現する
4. 季節感・限定感・トレンド感のいずれかを盛り込む

## ファネル・導線設計

### 基本ファネル

```
コレクション広告（全商品が一覧で見える）
  │
  ├─ サムネイル①クリック → 商品ページA → カート → 購入
  ├─ サムネイル②クリック → 商品ページB → カート → 購入
  ├─ サムネイル③クリック → 商品ページC → カート → 購入
  ├─ サムネイル④クリック → 商品ページD → カート → 購入
  └─ サムネイル⑤クリック → 商品ページE → カート → 購入
                                                │
                                          サンクスページ
                                        （X Pixel コンバージョン計測）
```

### 導線設計のポイント

1. **サムネイル → 個別商品ページ**: 各サムネイルから直接その商品の詳細ページへ遷移させる
2. **商品ページ → カート → 購入**: ECサイトの標準的な購入フローに接続する
3. **サンクスページでX Pixelのコンバージョンイベントを発火**: Purchase イベントを計測する
4. **DPA（ダイナミックプロダクト広告）との連携**: コレクション広告の閲覧者に対してDPAでリターゲティングを実施し、閲覧した商品を再度表示する

### DPA連携によるリターゲティング

コレクション広告で商品一覧を見たが購入に至らなかったユーザーに対し、DPAで閲覧商品を再表示することで購入を促進する。X Pixelの Content View / Add to Cart イベントデータを活用する。

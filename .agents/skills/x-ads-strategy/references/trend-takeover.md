# Trend Takeover / Trend Takeover+（トレンドテイクオーバー / トレンドテイクオーバープラス）

## フォーマット概要

Exploreタブのトレンドセクションに表示される広告。**Trend Takeover+（Spotlight）** はトレンドリストの上部にリッチなビジュアル（動画または静止画）を追加表示する上位版。1日1カ国1広告主の独占枠。

---

## 入稿スペック

### Trend Takeover

| 項目 | 仕様 |
|------|------|
| トレンド名 | 最大20文字 |
| トレンド説明文 | 最大30文字（任意だが推奨） |
| コンパニオン広告 | 3〜6本必須（画像/動画/GIF） |

### Trend Takeover+（Spotlight）

| 項目 | 仕様 |
|------|------|
| フォーマット | MP4動画 または 静止画 |
| アスペクト比 | 16:9 |
| 推奨解像度 | 1280x720 |
| ファイルサイズ | 画像：5MB以下 / GIF：15MB以下 |
| コンテンツ制限 | 全年齢対象コンテンツ必須（武器・ヌード・暴力表現は不可） |
| 配信形式 | ダークポスト配信（メディアのみ、カードなし） |

## ベストプラクティス

### Trend Takeover クリエイティブ
- **トレンド名は20文字以内**でキャッチーなフレーズを設計する
- **説明文は30文字以内**でトレンドの文脈を補足する
- **コンパニオン広告を3〜6本**用意し、トレンドクリック後のタイムラインを充実させる
- トレンド名はユーザーがクリックしたくなる好奇心トリガーを含める

### Trend Takeover+ クリエイティブ
- **16:9の動画または静止画**がExploreタブ上部を占有する
- **全年齢対象コンテンツが必須**：武器・ヌード・暴力表現は不可
- ダークポスト配信（メディアのみ、カードなし）で表示される
- 一目でブランド/キャンペーンが分かるビジュアルインパクトを最優先

### トレンド名設計のコツ
1. 短く覚えやすいフレーズにする（20文字上限を活用）
2. 好奇心を刺激する問いかけや宣言型の表現
3. ブランド名を含めるか、キャンペーン固有のワードを使う
4. ハッシュタグとして機能するフレーズ設計

## コピー構成

- **トレンド名がコピーそのもの**：記憶に残るフレーズ設計が最重要
- ユーザーのクリックを誘発する好奇心トリガーを組み込む
- コンパニオン広告のコピーでトレンドの文脈を補足・展開する

### コピーの階層構造
```
[トレンド名（20文字以内）] ← 最重要：記憶に残るフレーズ
  + [トレンド説明文（30文字以内）] ← 文脈補足
    + [コンパニオン広告コピー ×3〜6] ← 詳細訴求・展開
```

### トレンド名の例
```
「#新時代のはじまり」
「#まだ知らないの」
「#[ブランド名]革命」
```

## ファネル・フロー設計

```
Trend Takeover（Exploreタブでの発見）
  → クリック
    → トレンドハッシュタグタイムライン（UGC含む）
      → コンパニオン広告（3〜6本）
        → 商品LP / キャンペーンサイト
```

### ファネル設計のポイント

#### ハッシュタグキャンペーンとの連動
- **注意**：2025年6月以降、広告本文中のハッシュタグは禁止されているが、トレンド名のハッシュタグは引き続き使用可能
- オーガニック投稿やUGCでハッシュタグキャンペーンを並行展開し、トレンドタイムラインを盛り上げる

#### イベント・文化的モーメントとの連携
- イベントやカルチャーモーメント（祝日、スポーツ大会、音楽イベントなど）のタイミングに合わせた配信が効果を最大化する
- トレンドテイクオーバーの効果はタイミングの精度に大きく依存する

#### Timeline Takeover との同日併用
```
[Trend Takeover] Exploreタブをジャック
  + [Timeline Takeover] ホームタイムラインをジャック
    = Xプラットフォーム全体での最大インパクト
```

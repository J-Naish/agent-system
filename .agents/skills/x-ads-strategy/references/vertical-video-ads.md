# X広告フォーマット：縦型動画広告

---

## フォーマット概要

縦型動画広告はフルスクリーンで表示され、音声ONがデフォルトのフォーマットである。X広告の中で最も急成長している配信面であり、CTAボタンが再生開始1秒後に自動表示される。iOS/Androidのモバイルデバイスのみで配信される。

TikTokやInstagram Reelsと同じ9:16フォーマットであるため、クロスプラットフォームでのクリエイティブ転用が容易だが、X固有のルールへの調整が必要である。

## クリエイティブベストプラクティス

### セーフゾーン設計

フルスクリーン表示のため、UIオーバーレイに重要な情報が隠れないよう注意が必要。

| エリア | 注意事項 |
|--------|---------|
| 画面上部 | UIが2秒間表示される → 重要な情報を配置しない |
| 画面下部 | CTAボタンが1秒後に自動表示 → テキストやロゴを被せない |
| 中央60% | 重要なコンテンツはこのエリアに集中させる |

### 音声ON前提の設計

縦型動画はフルスクリーン表示のため、音声ONがデフォルトとなる。

- **ナレーション・BGMありきで設計**：音声をメインの情報伝達手段として活用
- **冒頭にインパクトのあるサウンド**：音でもスクロールを止める設計
- **字幕は引き続き追加**：公共の場でミュートにしているユーザーもいるため、字幕は保険として必ず付ける

### TikTok/Reelsからの転用ルール

9:16フォーマットが共通のため、TikTokやInstagram Reelsのクリエイティブを転用できるが、以下の調整が必要。

| 調整項目 | 詳細 |
|---------|------|
| ハッシュタグの削除 | X広告ではハッシュタグの使用が禁止されている。TikTok向けに入れたハッシュタグは全て削除する |
| 絵文字の制限 | 最大1つまで。TikTok/Reelsで多用した絵文字は削減する |
| TikTokウォーターマーク削除 | TikTokのロゴ透かしが入った動画は必ず除去する |
| X向けCTAボタン設計 | XのCTAウェブボタンに合わせた導線設計に変更する |

## 広告コピー構成

### コピーの位置づけ

縦型動画広告ではフルスクリーンの動画が主役であり、コピーの重要度は相対的に低い。

### コピー設計原則

- **30〜50文字で動画のコンテキストを補足**：コピーは動画の背景情報を伝える程度に留める
- **CTAはウェブサイトボタンに委ねる**：コピー内にCTAを書く必要はない
- 動画がメッセージの全てを担い、コピーはあくまで補助的な役割

### コピー構成

```
[動画の文脈を補足する短いテキスト（30〜50文字）]
  ↓
CTA → ウェブサイトボタン（1秒後に自動表示）
```

## ファネル/フロー設計

### 基本ファネル構造

```
リーチ目的の縦型動画広告
  ↓
広いリーチで大量のインプレッションを獲得
  ↓
視聴者をリターゲティングオーディエンスとして蓄積
  ↓
コンバージョン広告（画像/動画+ウェブサイトカード）でリターゲティング
  ↓
LP → コンバージョン
```

### タイムライン広告との組み合わせ

縦型動画広告はタイムライン広告と併用することで、追加で+10%のインクリメンタルインプレッションを獲得できる。

```
【並行配信】
縦型動画広告（フルスクリーン配信面）
  +
タイムライン広告（フィード配信面）
  ↓
2つの配信面で同時にリーチを拡大
  ↓
インクリメンタルインプレッション +10%
```

### ブランドセーフティ

IAS Pre-Bid（事前入札フィルタリング）が利用可能であり、ブランドセーフティ違反がゼロの実績がある。安心して大規模な配信が可能。

### 推奨ファネルフロー

```
Step 1: リーチ目的 → 縦型動画で広範囲に認知獲得
  ↓
Step 2: タイムライン広告と並行配信 → インプレッションの最大化
  ↓
Step 3: 視聴者データを蓄積 → リターゲティングオーディエンス構築
  ↓
Step 4: コンバージョン広告 → ウェブサイトカード付き動画/画像で刈り取り
  ↓
Step 5: LP → コンバージョン
```

## まとめ

縦型動画広告はX広告で最も成長著しいフォーマットであり、フルスクリーン・音声ONの没入的な視聴体験を提供する。タイムライン広告と比較してエンゲージメント7倍、CPM-14%という優れたパフォーマンスを発揮する。TikTok/Reelsからのクリエイティブ転用が可能だが、ハッシュタグ削除・絵文字制限・ウォーターマーク除去のX固有ルールへの対応が必須である。リーチ目的で広範囲に配信し、視聴者をリターゲティングする戦略が最も効果的である。

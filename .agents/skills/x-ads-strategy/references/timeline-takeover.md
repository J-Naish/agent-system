# Timeline Takeover（タイムラインテイクオーバー）

## フォーマット概要

その日ユーザーがXを開いた際に最初に目にする広告。タイムライン上で最もプレミアムな配置。1日単位の独占枠であり、最大級のリーチとインパクトを実現する。

---

## 対応フォーマット

以下のクリエイティブフォーマットに対応：

- 画像（Image）
- 動画（Video）
- カルーセル（Carousel）
- X Live
- カンバセーションボタン（Conversation Button）
- 投票（Poll）
- ブランド通知（Branded Notification）

---

## 配信面・目的・課金

| 項目 | 詳細 |
|------|------|
| 配信面 | ホームタイムラインの最上部（その日の最初の広告枠） |
| 広告目的 | Reach（リーチ）、Video Views（動画再生数） |
| 課金方式 | 日額固定（推定$200K〜$250K+/日、市場により変動） |
| フリークエンシー | 最大3回/日（24時間） |

| 指標 | 効果 |

| キャンペーン認知 | 3倍 |

## ベストプラクティス

### クリエイティブ戦略
- **動画フォーマットを強く推奨**：静止画よりも圧倒的に注目度が高い
- インパクトファーストのビジュアル設計
- **「ステートメント型」クリエイティブ**：ブランドの世界観を一発で伝える
- タイムラインのフォローアップ用に**3〜6本のコンパニオン広告**を準備する

### クリエイティブ制作の原則
1. 最初の1秒で視線を奪うビジュアルインパクト
2. ブランドの新しい方向性・製品・イベントを宣言する構成
3. 音声OFFでも伝わるデザイン（テキストオーバーレイ活用）
4. コンパニオン広告との統一感のあるビジュアルトーン

---

## コピー構成

- **宣言型コピー**：大型ローンチ、新方向性、新製品、イベント告知に最適
- 新しい方向性・製品・イベントを簡潔な言葉で表現する
- **ワンラインメッセージで十分**：短く力強い一文が最も効果的

### コピーの基本構造
```
[宣言型ヘッドライン（1行）]
+ ブランド名 / ハッシュタグ
```

### コピー例
```
「新しい[ブランド名]、はじまる。」
「[製品名]、ついに登場。」
「[イベント名] × [ブランド名]」
```

---

## ファネル・フロー設計

```
Timeline Takeover（ファーストインパクト）
  → 同日中のコンパニオン広告（3〜6種類をタイムラインで展開）
    → 翌日以降、リーチしたユーザーに通常のプロモーテッド広告でリターゲティング
```

### ファネル設計のポイント
- **Trend Takeover との同日併用**で最大効果を実現する
- テイクオーバー当日はオーガニック投稿（自社アカウントの通常投稿）も連動させる
- コンパニオン広告は製品詳細・ユーザーボイス・限定オファーなどバリエーションを持たせる
- 翌日以降のリターゲティングでコンバージョン導線を構築する

### 同日施策の全体像
```
[Timeline Takeover] メイン広告（朝〜）
  + [コンパニオン広告 ×3〜6] タイムライン内で展開
  + [オーガニック投稿] 自社アカウントから連動発信
  + [Trend Takeover] 同日併用でExploreタブもジャック（オプション）
```

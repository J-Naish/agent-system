# Amplify Pre-roll（アンプリファイ・プレロール）

## フォーマット概要

200以上のコンテンツパートナーが提供するプレミアム動画コンテンツの再生前に表示される動画広告。15以上のカテゴリから配信先を選択可能。6秒後にスキップボタンが表示される。セルフサーブ（自社運用）対応。

---

## 入稿スペック

| 項目 | 仕様 |
|------|------|
| ファイルサイズ | 最大1GB |
| 推奨尺 | 15秒以下 |
| 最大尺 | 2分20秒 |
| ファイル形式 | MP4 / MOV |
| 推奨アスペクト比 | 1:1（9:16、16:9もサポート） |
| 推奨解像度 | 1200x1200（1:1の場合、最小600x600） |
| スキップボタン | 6秒後に表示 |

---

## カテゴリ・配信面・目的

| 項目 | 詳細 |
|------|------|
| コンテンツカテゴリ | スポーツ、音楽、エンターテインメント、イベント、ニュースなど（15カテゴリ以上） |
| 配信面 | プレミアムパブリッシャーの動画コンテンツ再生前（インフィード） |
| 広告目的 | Pre-roll Views（プレロール再生数専用目的） |
| 課金方式 | CPV（50%以上の可視性 + 6秒以上視聴、または拡大/ミュート解除） |
| 入札方式 | 最大入札（CPV上限設定）またはオート入札（推奨：最低価格での最適化） |

---

## ベストプラクティス（X公式6項目）

### 1. 6秒以内に収める
88%の消費者が15秒以下の広告を求めている。短尺ほど完了率が高い。

### 2. コアメッセージは1つに絞る
複数メッセージを詰め込むと完了率が-13%低下する。

### 3. 関連コンテンツカテゴリをターゲティングする
関連性の高いコンテンツの前に配信すると、ブランド好感度が+40%向上する。

### 4. ブランドを常に表示する
ロゴを常時表示することで、広告想起が+60%向上する。

### 5. 音声OFFを前提にデザインする
テキストオーバーレイ＋ビジュアルで伝わるクリエイティブにする。

### 6. 他の動画広告と組み合わせる
プレロール＋インフィード動画を併用すると、広告想起が+19%向上する。

## スキップ対策

| シナリオ | 対策 |
|----------|------|
| 全般 | コアメッセージを最初の5秒以内に配置する |
| 6秒以下の動画 | スキップボタン表示前に動画が終了するため、スキップされない |
| 6秒超の動画 | 6秒時点までに商品＋ベネフィットが伝わる構成にする |

---

## コピー構成

- プレロール広告にはポストテキスト（投稿文）は付かない（動画のみ表示）
- 動画内のテキストオーバーレイがコピーの役割を果たす
- URL（ランディングページ）はオプションで設定可能

### 動画内テキスト設計の原則
- 1〜3秒：ブランドロゴ＋フック（注意喚起）
- 3〜5秒：コアメッセージ＋ベネフィット
- 5〜6秒：CTA（行動喚起）またはブランド名リマインド

---

## ファネル・フロー設計

```
プレロール（6秒）
  → ブランド認知の獲得
    → 視聴者をオーディエンスとして蓄積
      → インフィード動画広告（詳細訴求）
        → ウェブサイトカード広告
          → LP → コンバージョン
```

### ファネル設計のポイント
- ブランドに関連性の高いコンテンツカテゴリを選択する
- 他の動画広告フォーマットとの組み合わせで広告想起が+19%向上する
- プレロールは認知獲得レイヤーとして位置づけ、詳細訴求は後続フォーマットに任せる

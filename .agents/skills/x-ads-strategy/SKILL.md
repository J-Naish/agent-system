# X Ads 戦略・設計スキル

X（旧Twitter）広告の戦略立案から仕様書作成までを一貫して支援する。15種類の広告フォーマットから最適な組み合わせを選定し、Aesthetic Scoreを最大化するクリエイティブ設計と運用設計を行う。

## ワークフロー

```
Step 1: 情報収集         → ビジネス情報・目的・予算・ターゲットを整理
    ↓
Step 2: 広告フォーマット選定 → 15種類から目的・予算に最適なフォーマットを選定
    ↓
Step 3: 戦略・クリエイティブ設計 → キャンペーン構造・ターゲティング・クリエイティブを設計
    ↓
Step 4: 仕様書作成        → 運用仕様書・クリエイティブ仕様書を出力
```

## Step 2: 広告フォーマット選定

X広告は **セルフサーブ（運用型）** と **マネージドサービス（予約型）** の2つの購入方式がある。

| 方式 | 特徴 | 適したケース |
|-----|------|-------------|
| セルフサーブ | Ads Managerから自由に出稿、柔軟な予算調整 | パフォーマンス重視、テスト、中小予算 |
| マネージド | X営業担当経由、固定価格、配信保証 | ブランディング、大規模リーチ、プレミアム枠 |

### インタラクティブ機能（付加機能）

以下の機能は画像広告・動画広告と組み合わせて使用する。各フォーマットのリファレンスに詳細を記載。

| 機能 | 説明 | 主な仕様 |
|------|------|---------|
| カンバセーションボタン | ハッシュタグ付き投稿を促す | 事前入力256文字、ハッシュタグ21文字 |
| 投票（Poll） | 2〜4つの選択肢で投票 | 各25文字/12文字（日）、5分〜7日 |
| アプリカード | App Store/Play Storeリンク | CTA: Install, Open, Play, Shop等 |
| ウェブサイトカード | LPへのクリック導線 | 見出し最大70文字（50文字推奨） |

### 各フォーマットの活用シーン

**テキスト広告** — メッセージのA/Bテストやシンプルな告知に。低予算で複数のコピーバリエーションを検証し、勝ちコピーを画像/動画広告に展開する起点として使う。

**画像広告** — 最も汎用的なフォーマット。商品ビジュアル訴求、Webサイト誘導、アプリインストールに。ウェブサイトカード/アプリカード/投票/カンバセーションボタンと組み合わせ可能。

**動画広告** — ブランドストーリーテリング、商品デモ、感情的キャンペーンに。15秒以下+キャプション+3秒フック+ロゴ常時表示でエンゲージメント+33%。全9目的に対応する最も柔軟なフォーマット。

**バーティカルビデオ広告** — X上で最もエンゲージメントが高い。全画面・音声ONの没入型体験。タイムライン広告比でエンゲージメント7倍、CPM14%低い。TikTok/Reelsクリエイティブの転用が可能。

**カルーセル広告** — 複数商品のショーケース、連続ストーリーテリング、機能紹介に。2〜6枚の画像/動画。シングル/マルチデスティネーション対応。アプリインストール+24%。

**コレクション広告** — EC特化。ヒーロー画像+サムネイルで全商品を一覧表示（スワイプ不要）。CTR+42%。カルーセルとの違い：全商品が一度に見える。

**DPA（ダイナミックプロダクト広告）** — ユーザー行動に基づく自動商品表示。リターゲティングとプロスペクティングの2タイプ。CPA改善30〜88%。X Pixel + 商品カタログが必須。

**検索キーワード広告** — ユーザーの検索意図を捉えるフォーマット。Google検索広告のようなインテントマーケティングをX上で実現。2025年新設のKeywords目的を使用。最小25キーワード/キャンペーン。

**Amplifyプレロール** — 200以上のコンテンツパートナーの動画前に再生。6秒後にスキップ可能。15以上のカテゴリから選択。関連カテゴリ選択でブランド好感度+40%。

**Amplifyスポンサーシップ** — 特定パブリッシャーとの1:1独占提携。マネージドサービスのみ。大型イベントパートナーシップに最適。

**タイムラインテイクオーバー** — 1日の最初に表示される最もプレミアムな広告枠。ブランド認知2倍、キャンペーン認知3倍。大型ローンチ向け。

**トレンドテイクオーバー** — Exploreタブのトレンド欄に広告掲載。メッセージ関連付け3倍、好感度9倍。文化的モーメント連動に最適。

**Xライブ** — リアルタイムライブ配信。商品ローンチ、イベント中継、Q&Aに。テイクオーバーとの併用で最大効果。

**ブランド通知** — オプトイン式の通知配信。スケジュール/サブスクリプション/インスタントの3タイプ。商品ローンチカウントダウンやリマインダーに最適。

**ブランドハッシュタグ** — ブランド専用ハッシュタグにカスタム絵文字を付与。大型キャンペーン向けのプレミアム商品。

### ビジネス目標別の推奨構成

| ビジネス目標 | 推奨フォーマット | 推奨目的 |
|------------|---------------|---------|
| ブランド認知最大化 | バーティカルビデオ + 動画 | Reach |
| 商品認知 | 画像 + カルーセル | Reach / Video Views |
| Webサイト誘導 | 画像（Webカード付き）+ 動画（Webカード付き） | Website Traffic |
| アプリインストール | 動画（アプリボタン付き）+ カルーセル | App Installs |
| EC売上 | DPA + コレクション広告 | Sales |
| リード獲得 | 画像/動画（Webカード付き）+ カンバセーションボタン | Website Traffic / Sales |
| エンゲージメント | 投票付き広告 + カンバセーションボタン | Engagements |
| イベントプロモ | Xライブ + タイムラインテイクオーバー | Reach / Video Views |
| 検索意図の捕捉 | 検索キーワード広告 | Keywords |

### 予算別の推奨

| 月予算 | 推奨構成 |
|-------|---------|
| 〜5万円 | テキスト or 画像広告（1キャンペーン、メッセージテスト） |
| 5〜20万円 | 画像 + 動画（2キャンペーン、テスト→スケール） |
| 20〜50万円 | 上記 + バーティカルビデオ or カルーセル |
| 50〜100万円 | 上記 + DPA or コレクション（目的別に追加） |
| 100万円+ | プレミアム枠（Amplify/テイクオーバー）+ セルフサーブの併用 |

### 選定後のアクション

広告フォーマットを選定したら、以下をユーザーに提示して合意を取る：

1. 選定したフォーマットとその理由
2. 各フォーマットの想定予算配分
3. キャンペーン目的と最適化イベント
4. 想定KPI（CPM / CPC / CPA / ROAS の目安）

## Step 3: 戦略・クリエイティブ設計

キャンペーン構造、ターゲティング、入札、クリエイティブを設計する。詳細は以下のリファレンスを参照。

- キャンペーン構造・ターゲティング・入札・予算・計測 → [campaign-strategy.md](references/campaign-strategy.md)
- クリエイティブ原則・コピーフレームワーク・CTA・テスト → [creative-fundamentals.md](references/creative-fundamentals.md)
- 各フォーマット固有の設計 → Step 2 の各リファレンスファイル

### 3-1. キャンペーン構造設計

- **3階層構造**: キャンペーン（目的）→ 広告グループ（ターゲティング・予算、最大100個）→ 広告（クリエイティブ、無制限）
- **キャンペーン目的の選定が最重要**: 9種類（Reach / Video Views / Pre-roll Views / App Installs / Website Traffic / Engagements / Keywords / Sales / App Re-engagement）
- **CBO（キャンペーン予算最適化）**: 広告グループ間の予算を自動分配
- **学習期間**: 約10コンバージョン後に活性化、3〜5日。Sales目的は4〜6週間のデータ蓄積が必要

### 3-2. ターゲティング設計

- **属性ターゲティング**: 地域・言語・端末・年齢・性別
- **行動ベース**: 会話・イベント・キーワード（25以上推奨）・映画TV・興味関心・フォロワー類似
- **カスタムオーディエンス**: フォロワー、リスト（メール/電話）、Pixel訪問者、投稿エンゲージャー
- **推奨**: AIドリブン（属性のみ）か、キーワード30+ × フォロワー類似10アカウント+

### 3-3. クリエイティブ設計の原則

**Aesthetic Score** — 2025年導入のクリエイティブ品質評価。高スコアで最大20% CPM割引。

- ハッシュタグ禁止（2025年6月〜）
- 絵文字は最大1個（日本・韓国は例外的に柔軟）
- URLを本文に書かない（ウェブサイトカードを使用）
- クリーンで洗練されたデザイン
- 文法・句読点を正確に

**クリエイティブの基本ルール:**
- 3つ以上のフォーマット併用で認知度+20%、購入意向+7%
- 広告グループごとに5つ以上のユニークなクリエイティブ
- 1広告=1メッセージに集中
- ロゴは左上に常時表示
- 3〜4週間ごとにクリエイティブリフレッシュ

### 3-4. 命名規則

```
[プラットフォーム]_[目的]_[ターゲット]_[フォーマット]_[日付]

例:
キャンペーン: X_Sales_JP-25to34_Video_202602
広告グループ: AG_Retargeting_SiteVisitors_7d
広告:         Ad_ProductA_15sVideo_v1
```

**UTMパラメータ:**
```
utm_source=x
utm_medium=paid_social
utm_campaign={campaign_name}
utm_content={ad_name}
utm_term={keyword}（キーワード広告の場合）
```

## Step 4: 仕様書作成

Step 1〜3の設計内容を以下の形式でドキュメントとして出力する。

### 出力物

1. **戦略概要書** — ビジネス目標、広告フォーマット選定理由、キャンペーン構造、予算配分、KPI目標
2. **キャンペーン設計書** — キャンペーン×広告グループ×広告の具体的な設定値一覧（目的、ターゲティング、入札、スケジュール、命名規則）
3. **クリエイティブ仕様書** — フォーマット別のクリエイティブ仕様（サイズ、テキスト、CTA）とコピー案
4. **計測設計書**（必要に応じて） — X Pixel / CAPI のイベント設計、アトリビューションウィンドウ、UTMパラメータ

### 出力フォーマット

各出力物はマークダウン形式で、`contexts/strategy/` ディレクトリに保存する。

### 日本市場の特徴

- 匿名文化：複数アカウント（本垢・裏垢）が一般的
- 利用傾向：情報収集、トレンドチェック、エンタメ消費
- 言語ターゲティング注意：日本語設定でオーディエンス約20%減少の場合あり
- 動画広告のCPMが低くコスト効率が良い
- 絵文字は例外的に複数使用が許容される場合がある
- 規制：景表法、薬機法、ステマ規制（2023年10月施行）

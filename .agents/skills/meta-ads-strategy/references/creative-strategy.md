## 1. クリエイティブの重要性（2025〜2026年）

Meta広告のパフォーマンスにおいて、クリエイティブの品質は最も重要な要素。

- 広告パフォーマンスの **70〜80%** はクリエイティブの強さに依存（予算やターゲティングではない）
- 2026年の Meta 広告は「**80%がクリエイティブ運用、20%がメディアバイイング**」
- Andromeda アルゴリズムにより、Meta は広告クリエイティブの内容から配信先を判断する

## 2. クリエイティブ多様性 vs ボリューム

### 原則: 多様性 > 量

2026年のアルゴリズムは、類似した大量のバリエーションではなく、**根本的に異なるクリエイティブコンセプト**を評価する。

**NG パターン:**
- 1つの勝ちパターン画像にフックや見出しを20パターン付ける → アルゴリズム上逆効果

**OK パターン:**
- 勝ちパターンのメッセージを**異なるクリエイティブフォーマット**で再利用する
- 根本的に異なるアングル・トーン・クリエイティブ手法を展開

### クリエイティブの分類

広告ライブラリは以下の多様なクリエイティブを含むべき:

| カテゴリ | 例 |
|---------|------|
| 静止画 | 商品写真、テキストオーバーレイ画像、インフォグラフィック |
| 短尺動画（Raw/Lo-fi） | スマホ撮影、UGC風、テスティモニアル |
| 制作動画（Polished） | シネマティック、モーショングラフィックス |
| 創業者/スタッフ出演 | セルフィー動画、インタビュー |
| GIF/ミーム | カジュアルな訴求 |
| カルーセル | 複数商品、ステップ解説、ビフォーアフター |

### クリエイティブコンセプトの例

- **シネマティックなブランドストーリー** — ブランドの世界観を伝える
- **ミーム風の比較** — 親しみやすいユーモアで差別化
- **Lo-fi テスティモニアル** — リアルなユーザーの声
- **モーショングラフィックス** — データやプロセスの視覚化
- **ノスタルジック系** — 懐かしさを活用した感情訴求

## 3. クリエイティブの設計ルール

### 配置面別のクリエイティブ設計

**設計の鉄則:** 9:16（縦型フルスクリーン）をマスターデザインとし、Reelsのセーフゾーン（下部35%を空ける）に合わせれば、Stories・Feed含む全配置面で安全に表示される。

| 配置面 | アスペクト比 | セーフゾーン | 設計のポイント |
|--------|------------|------------|---------------|
| **Feed** | 4:5（推奨）、1:1 | 上下左右100px | 4:5は1:1より最大15%パフォーマンス向上 |
| **Stories** | 9:16 | 上270px、下380px、左右65px | CTA/ロゴを中央に配置。上下を避ける |
| **Reels** | 9:16 | 上270px、**下670px**、左右65px | 下部35%が最も広い制限。ここを基準に設計 |
| **In-Stream** | 16:9 | 中央80%以内 | 横型動画。デスクトップ視聴が多い |
| **Marketplace/Search** | 1:1 | 中央80%以内 | 商品画像中心のシンプルな構成 |

### 動画の設計

| 要素 | 推奨事項 |
|------|---------|
| 最初の3秒 | フックで注目を獲得する（最重要） |
| 長さ | フィード: 15〜60秒、リールズ: 15〜30秒、ストーリーズ: 最大15秒 |
| アスペクト比 | 9:16（縦型）が主流（Meta広告在庫の90%）。フィードは4:5 |
| 字幕 | 必須（ユーザーの80%は音声OFFで視聴） |
| トーン | シリアス・インタビュー・ドラマ系のフックが、遊び心のある形式よりパフォーマンスが高い |
| CTA | 明確なアクション指示を入れる |
| UGC風 | UGC風の動画は通常の広告より4倍のエンゲージメント |

### 静止画の設計

| 要素 | 推奨事項 |
|------|---------|
| アスペクト比 | 4:5（フィードで最大15%パフォーマンス向上）、1:1、9:16 |
| 解像度 | 1080×1350px（4:5）、1080×1080px（1:1）以上。Retina対応は1440px以上 |
| メッセージ | 1つの明確なオファー/訴求に絞る |
| テキスト量 | 画像内テキストは最小限（20%ルールは撤廃されたが少ない方が良い） |
| 品質 | 高品質な写真/グラフィックが必須 |
| CVへの貢献 | 全体CVの60〜70%は依然として静止画から発生。動画偏重にしない |

### テキスト（コピー）の設計

テスト対象として、以下の長さのバリエーションを用意:

| 長さ | 特徴 |
|------|------|
| 超短文 | 1〜2行。スクロールを止める一言 |
| 中文 | 3〜5行。課題と解決策 |
| 長文 | パラグラフ形式。詳細な説明やストーリー |
| ブログ長文 | 長編コピー。情報量で説得 |

### 訴求軸のバリエーション

異なるモチベーションに訴えるクリエイティブを用意:

| 訴求軸 | 内容 |
|--------|------|
| ペインポイント | 「〜で困っていませんか？」 |
| プレジャー/ベネフィット | 「〜が手に入ります」 |
| テスティモニアル/社会的証明 | 「お客様の声」「導入実績」 |
| 好奇心 | 「なぜ〜は〜なのか？」 |
| 直接質問 | 「〜をお探しですか？」 |
| 限定/緊急性 | 「今だけ」「残りわずか」 |
| 比較 | 「従来の方法 vs 私たちの方法」 |

## 4. クリエイティブテストの方法

### 3-3-3フレームワーク

ファネルの各段階に合わせたクリエイティブを設計:

| 段階 | メッセージの方針 | 前提 |
|------|-----------------|------|
| **ToF（認知）** | 問題提起。ブランド認知ゼロを前提 | ユーザーは商品を知らない |
| **MoF（検討）** | 解決策の評価を支援 | ユーザーは課題を認識し解決策を比較中 |
| **BoF（購入）** | 限定オファー、社会的証明、保証、競合比較 | ユーザーは購入検討段階 |

### テストの運用サイクル

推奨される週次サイクル:

1. **5〜10本の新しい広告を投入**
2. **7日間、変更なしで配信**
3. **パフォーマンス不良の広告を停止、勝者をスケール**
4. **パフォーマンスデータからインサイトを抽出**
5. **インサイトを元に次のクリエイティブコンセプトを生成**
6. **1に戻る**

### 勝者の判断基準

| 指標 | 見方 |
|------|------|
| CTR | クリエイティブの引きの強さ |
| CPA / ROAS | ビジネス目標に対する効率 |
| フリクエンシー | 3.0以上は疲労の兆候 |
| CPMr（1000リーチあたりコスト） | $20以下が健全。上昇は疲労のサイン |
| フックレート（動画3秒視聴率） | 動画の冒頭の引きの強さ |

### クリエイティブの寿命管理

- **30〜60日** で勝ちクリエイティブもリフレッシュを検討
- **フリクエンシー上昇、CTR低下** が疲労のサイン
- 疲労時は入札調整ではなく**クリエイティブのリフレッシュ**が正解
- 月に最低 **2〜4回** はクリエイティブを更新

## 5. ファネル段階別のクリエイティブ戦略

### ToF（認知 / Upper Funnel）

- **目的:** 新規オーディエンスの教育、ペインポイントと欲求に語りかける
- **フォーマット:** 動画（リールズ）、画像、カルーセル
- **コンテンツ:** ブランドストーリー、問題提起、教育コンテンツ
- **トーン:** 情報的、感情的、共感的

### MoF（検討 / Middle Funnel）

- **目的:** ブランドを認識したユーザーを購入検討へ導く
- **フォーマット:** カルーセル、動画、テスティモニアル
- **コンテンツ:** ケーススタディ、比較、詳細な商品説明、FAQ
- **トーン:** 信頼構築、論理的、具体的

### BoF（購入 / Lower Funnel）

- **目的:** 購入検討中のユーザーを購買行動に転換
- **フォーマット:** 動的商品広告、コレクション、画像
- **コンテンツ:** 限定オファー、クーポン、テスティモニアル、保証、無料トライアル
- **トーン:** 緊急性、確信、行動喚起

## 6. Partnership Ads（パートナーシップ広告）

### 概要

クリエイターやインフルエンサーのアカウントから広告を配信する形式。

### メリット

- クリエイターの既存オーディエンスとページ学習データを活用
- 信頼性の構築（ブランドアカウントより自然な印象）
- より効率的なオーディエンス発見
- ブランドの広告ハンドルよりもパフォーマンスが高い傾向

### 推奨ケース

- UGC戦略を強化したい場合
- 新規市場・新規ターゲットにリーチしたい場合
- ブランド広告で成果が頭打ちになった場合

## 1. 全フォーマット共通の原則

### 1-1. クリエイティブが成果の70〜80%を決める

2025〜2026年のMeta広告では、Andromedaアルゴリズムにより、クリエイティブの内容がターゲティングの代替として機能する。予算やターゲティングの調整よりも、**クリエイティブの品質と多様性**が最も重要な成功要因。

### 1-2. コピーの文字数制限

| 要素 | 推奨文字数 | 超過時の挙動 |
|------|-----------|-------------|
| **メインテキスト（Primary Text）** | 125文字以内 | 「もっと見る」リンクで折りたたまれる |
| **見出し（Headline）** | 27文字以内 | 超過分は非表示になる |
| **説明文（Description）** | 27文字以内 | 配置面により表示されない場合がある |

> **注意:** Instagramでは説明文が表示されないケースが多い。見出しとメインテキストに訴求を集中させる。

### 1-3. コピーのテスト方針

1つのクリエイティブに対して **3パターン以上のコピーバリエーション** を用意してテストする。

| 長さ | 特徴 | 適したケース |
|------|------|-------------|
| **超短文（1〜2行）** | スクロールを止める一言 | 認知度の高い商品、シンプルなオファー |
| **中文（3〜5行）** | 課題→解決策の提示 | 中価格帯の商品、既知のカテゴリ |
| **長文（1段落以上）** | 詳細な説明やストーリー | 高額商品、未知のカテゴリ、B2B |
| **ブログ長文** | 情報量で説得する長編 | 教育的コンテンツ、複雑なサービス |

### 1-4. フックの設計（最初の1行 / 最初の3秒）

フックはコピーの最初の1行（テキスト広告）または動画の最初の3秒に配置する。

#### フックの16タイプ

| # | タイプ | 説明 | 例 |
|---|--------|------|-----|
| 1 | **質問型** | 思考を刺激する問いかけ | 「まだ〇〇で悩んでいますか？」 |
| 2 | **数字/統計型** | データで信頼性を訴求 | 「92%のユーザーが〇〇を実感」 |
| 3 | **ベネフィット型** | 得られる結果を直接提示 | 「たった3日で〇〇が変わる」 |
| 4 | **ペインポイント型** | 痛みに共感する | 「〇〇にうんざりしていませんか？」 |
| 5 | **ストーリー型** | 物語の冒頭で引き込む | 「去年まで私は〇〇でした。でも今は...」 |
| 6 | **FOMO型** | 緊急性・希少性を演出 | 「残り24時間」「限定100個」 |
| 7 | **逆説型** | 常識を覆す主張 | 「〇〇は実は逆効果です」 |
| 8 | **ビフォーアフター型** | 変化を視覚的に見せる | 変化前後の比較画像/動画 |
| 9 | **好奇心ギャップ型** | 不完全な情報で続きを見させる | 「プロが絶対にやらない3つのこと」 |
| 10 | **サプライズ型** | 予想外の展開で注目 | 意外な使い方、驚きの結果 |
| 11 | **社会的証明型** | 他者の評価で安心感 | 「★4.8 / 10,000件のレビュー」 |
| 12 | **直接呼びかけ型** | ターゲットを名指し | 「30代の〇〇にお勤めの方へ」 |
| 13 | **クリフハンガー型** | サスペンスで動画を最後まで見させる | 「結果は...（動画の最後で）」 |
| 14 | **挑発型** | 反論したくなる主張 | 「〇〇はもう時代遅れです」 |
| 15 | **インタラクション型** | 投票やクイズで参加を促す | 「A vs B、あなたはどっち？」 |
| 16 | **ASMR/映像美型** | 視覚的に魅了する | 商品の開封、調理過程、製造工程 |

#### フックレート（Hook Rate）のベンチマーク

| 水準 | フックレート（3秒視聴率） |
|------|------------------------|
| 平均 | 15〜20% |
| 良好 | 20〜40% |
| 優秀 | 40%以上 |

### 1-5. CTA（コールトゥアクション）の設計

#### CTAボタンの選び方

| CTAボタン | 適した目的 | 適したビジネスタイプ |
|----------|-----------|-------------------|
| **今すぐ購入（Shop Now）** | 購入促進 | EC / D2C |
| **詳しくはこちら（Learn More）** | 情報提供・教育 | 全般（最も汎用的） |
| **登録する（Sign Up）** | リード獲得 | SaaS、メルマガ、ウェビナー |
| **予約する（Book Now）** | 予約促進 | サロン、クリニック、ホテル |
| **お問い合わせ（Contact Us）** | 問い合わせ獲得 | B2B、サービス業 |
| **ダウンロード（Download）** | アプリ/資料DL | アプリ、ホワイトペーパー |
| **申し込む（Apply Now）** | 申込促進 | 求人、金融、教育 |
| **見積もり（Get Quote）** | 見積もり依頼 | 保険、リフォーム、B2B |
| **チケット購入（Buy Tickets）** | チケット購入 | イベント、エンタメ |

**選定ルール:**
- CTAボタンは広告コピーの訴求内容と一致させる
- CTAボタンのリンク先はボタンの期待に合ったページにする
- CTAボタンありの広告は、なしの広告と比較してCTRが **2.85倍** 向上

### 1-6. 訴求軸（Ad Angle）の設計

| 訴求軸 | 内容 | コピー例 |
|--------|------|---------|
| **ペインポイント** | 課題・悩みに共感 | 「毎朝のメイクに30分かけていませんか？」 |
| **ベネフィット/USP** | 得られる価値を提示 | 「たった5分で完璧な仕上がり」 |
| **社会的証明** | 他者の評価・実績 | 「累計10万個突破。お客様満足度98%」 |
| **比較/差別化** | 競合や従来品との違い | 「従来品の3倍長持ち」 |
| **限定/緊急性** | 今すぐ行動する理由 | 「本日23:59まで。初回限定50%OFF」 |
| **好奇心** | 続きが気になる問いかけ | 「なぜプロは〇〇を使わないのか？」 |
| **ラグジュアリー/プレミアム** | 高品質・特別感 | 「一流シェフが認めた味」 |
| **スピード/手軽さ** | 手間がかからない | 「注文から最短翌日お届け。送料無料」 |

## 2. 画像広告（Image Ads）のクリエイティブ制作

### 2-1. 仕様

| 項目 | 仕様 |
|------|------|
| 推奨サイズ | 1080×1350px（4:5）、1080×1080px（1:1） |
| Retina対応 | 1440×1440px以上 |
| ファイル形式 | JPG、PNG |
| ファイルサイズ | 30MB以下（1MB以下推奨で読み込み高速化） |
| 画像内テキスト | 最小限に（20%ルールは撤廃されたが少ない方が配信効率が良い） |

### 2-2. デザインのベストプラクティス

**レイアウト原則:**
- **1つの広告 = 1つのメッセージ**: 複数の訴求を詰め込まない
- **4:5を優先**: フィードで1:1より最大15%高いパフォーマンス
- **高コントラスト**: フィードの中でスクロールを止めるために、太字・明確な色使い
- **ストック写真は避ける**: オリジナルの高品質写真が圧倒的に有効

**効果的なデザインパターン:**

| パターン | 内容 | 適したケース |
|---------|------|-------------|
| **商品ヒーロー** | 商品を大きく配置＋ベネフィットテキスト | EC、D2C商品 |
| **テキストオーバーレイ** | 太字のキャッチコピーを画像に重ねる | オファー訴求、セール告知 |
| **ビフォーアフター** | 使用前後の比較（分割画面） | 美容、ダイエット、リフォーム |
| **テスティモニアル** | 顧客の顔写真＋引用テキスト | 信頼構築、社会的証明 |
| **インフォグラフィック** | データや手順の視覚化 | 教育コンテンツ、B2B |
| **ライフスタイル** | 商品が使われているシーンの撮影 | ブランド認知、アパレル |
| **PNGオーバーレイ** | 商品切り抜きを背景に配置＋キャプション | 4:5動画風の静止画 |

**ブランド要素:**
- ロゴは控えめに配置（角の小さなロゴ、または色使いでブランドを表現）
- 一貫したデザインシステム（フォント、カラー、レイアウトの統一）で信頼感を構築

### 2-3. コピーの書き方

**メインテキスト（Primary Text）:**
```
フック: 1行目で注目を獲得（質問、数字、ペインポイント）
本文: ベネフィットを2〜3点簡潔に伝える
CTA: 「今すぐチェック↓」などの行動喚起
```

**見出し（Headline）:**
- 27文字以内で核心を伝える
- ベネフィット or オファーを直接的に表現

**フォーマットのテクニック:**
- 改行で読みやすくする
- 絵文字はブランドに合う場合のみ控えめに使用
- 箇条書きでベネフィットを列挙（✓ や → を使用）

## 3. 動画広告（Video Ads）のクリエイティブ制作

### 3-1. 仕様

| 項目 | 仕様 |
|------|------|
| 推奨サイズ | 1080×1920px（9:16）、1080×1350px（4:5）、1080×1080px（1:1） |
| 推奨長さ | フィード: 15〜60秒、リールズ: 15〜30秒、ストーリーズ: 最大15秒 |
| ファイル形式 | MP4、MOV |
| ファイルサイズ | 4GB以下 |
| コーデック | H.264、AAC音声128kbps以上 |
| フレームレート | 最大30fps |

### 3-2. 動画構成のフレームワーク

#### 基本構成: Hook → Body → CTA

```
[0〜3秒] フック — スクロールを止める
  ↓
[3〜15秒] ボディ — 価値を伝える（ベネフィット、デモ、ストーリー）
  ↓
[中間地点] 中間CTA — 行動を促す最初のポイント
  ↓
[残り] 追加情報 — 詳細、テスティモニアル、FAQ対策
  ↓
[最後] 最終CTA — 行動を再度促す
```

#### セーフゾーン（9:16の場合）

| 配置面 | 上部マージン | 下部マージン | 左右マージン |
|--------|------------|------------|------------|
| ストーリーズ | 270px（14%） | 380px（20%） | 65px（6%） |
| リールズ | 270px（14%） | **670px（35%）** | 65px（6%） |

> **鉄則:** リールズの下部35%を空けるデザインにすれば、ストーリーズ含む全配置面で安全に表示される。

### 3-3. デザインのベストプラクティス

**音声設計:**
- ユーザーの **80%は音声OFFで視聴** → 字幕は必須
- 大きく高コントラストな字幕で読みやすさを確保
- 音声ONの場合にも対応するBGM・ナレーションを用意

**映像設計:**
- 最初のフレームから動き（モーション）を入れる → スクロール停止効果
- 太字の白黒テキストキャプション＋グラフィックCTA（4:5動画で特に有効）
- 商品のPNGオーバーレイを動画に重ねる手法が高パフォーマンス
- トーンはシリアス/インタビュー/ドラマ系 > 遊び心のある形式

**クリエイティブタイプ別の推奨:**

| タイプ | 内容 | パフォーマンス特性 |
|--------|------|--------------------|
| **UGC風（Lo-fi）** | スマホ撮影の自然な映像 | エンゲージメント **4倍**。最も自然な印象 |
| **テスティモニアル** | 顧客が語るリアルな体験 | 信頼構築に最適。MoF/BoFで特に有効 |
| **創業者/スタッフ出演** | 顔出しのセルフィー動画 | ブランドの人間味を伝える |
| **デモ/ハウツー** | 商品の使い方を実演 | 理解促進。EC商品に最適 |
| **モーショングラフィックス** | アニメーション/データ視覚化 | B2B、SaaS、複雑なサービスに有効 |
| **シネマティック** | ブランドフィルム風の映像 | 認知キャンペーンのToF向け |

### 3-4. コピーの書き方

**メインテキスト:**
- 1行目で動画を見る動機を作る（動画の内容を予告 or 補完）
- 動画で伝えきれない詳細情報を補足
- 最終行にCTA

**見出し:**
- 動画の核心メッセージを27文字以内で凝縮
- 動画を見なくても訴求が伝わる自立したコピー

## 4. カルーセル広告（Carousel Ads）のクリエイティブ制作

### 4-1. 仕様

| 項目 | 仕様 |
|------|------|
| カード数 | 2〜10枚（推奨: 3〜5枚で開始） |
| 画像サイズ | 1080×1080px（1:1） |
| 動画サイズ | 1080×1080px（1:1）|
| 動画の長さ | 15秒以内推奨（最大240分） |
| 画像ファイルサイズ | 1MB以下推奨（最大30MB） |
| メインテキスト | 80文字（Facebook）/ 125文字（Instagram） |
| 見出し | カードごとに45文字以内 |
| アスペクト比 | 全カード統一必須（混在すると1枚目の比率にトリミングされる） |

### 4-2. カード構成のフレームワーク

#### 5ビート構成

```
カード1: フック — 注目を獲得し、スワイプを促す
カード2: コンテキスト — 背景・課題を提示
カード3: ディスカバリー — 解決策・商品を紹介
カード4: リワード — 証拠・テスティモニアル・詳細
カード5: CTA — 行動を促す最終カード
```

#### 効果的なカルーセル構成パターン

| パターン | カード構成 | 適したケース |
|---------|----------|-------------|
| **商品カタログ** | 商品A → 商品B → 商品C → ... | EC、複数商品の訴求 |
| **ベネフィット積み上げ** | 特徴1 → 特徴2 → テスティモニアル → オファー | SaaS、サービス |
| **ステップ解説** | Step1 → Step2 → Step3 → 結果 | ハウツー、プロセス紹介 |
| **ビフォーアフター** | Before → 使用中 → After → CTA | 美容、ダイエット、リフォーム |
| **ストーリーテリング** | 課題 → 出会い → 変化 → 結果 → CTA | ブランドストーリー |
| **FAQ/反論処理** | 疑問1 → 回答 → 疑問2 → 回答 → CTA | 検討段階のユーザー向け |

### 4-3. デザインのベストプラクティス

**カード1（ファーストカード）が最重要:**
- スワイプ率を決定するゲートキーパー
- 太字のベネフィット、好奇心ギャップ、驚きの統計、ビフォーアフターのティーザー
- 右端に矢印やオーバーラップする要素で「続きがある」ことを示す

**デザイン一貫性:**
- 全カードで背景色、フォント、写真スタイルを統一
- モバイルで読める高コントラスト・最小テキスト
- スワイプを促す視覚的キュー（矢印、カード間で連続する要素）

**カード順序の設定:**
- **ストーリーテリング型** → 「最もパフォーマンスの高いカードを先頭に」をOFFにして固定順序を維持
- **商品カタログ型** → 自動順序を有効にし、Metaのアルゴリズムに最適な順序を任せる

**パフォーマンスデータ:**
- カルーセル広告は単一画像広告と比較して **CPA 30〜50%低下**、**CPC 20〜30%低下**
- カードに価格を表示すると、ROAS向上につながるデータがある

### 4-4. コピーの書き方

**メインテキスト:**
- カルーセル全体のテーマや価値を1行で伝える
- スワイプする動機を作る

**各カードの見出し（45文字以内）:**
- カードごとに独立した訴求ポイント
- 同じメッセージの繰り返しは避ける

### 4-5. 避けるべきミス

- アスペクト比の混在（1枚目の比率で強制トリミングされる）
- 重すぎるファイル（読み込み速度低下）
- ストーリーテリング型なのに自動順序を有効にする
- フィード用デザインをそのままストーリーズに流用する（セーフゾーンが異なる）
- ほぼ同一のカードの繰り返し（アルゴリズムに多様性を提供できない）

## 5. コレクション広告（Collection Ads）のクリエイティブ制作

### 5-1. 仕様

| 項目 | 仕様 |
|------|------|
| カバー | 画像（1200×628px、16:9 or 1:1）または動画 |
| 商品画像 | Facebook: 4枚 / Instagram: 3枚（1:1で自動調整） |
| 見出し | 25文字以内 |
| 本文 | 90文字以内 |
| タップ時 | フルスクリーンのInstant Experienceに展開 |
| 前提条件 | Meta Business Managerの商品カタログ接続が必須 |
| 商品数 | Instant Experience内で最大50商品表示可能 |

### 5-2. デザインのベストプラクティス

**カバー画像/動画:**
- 最も目を引くヒーロービジュアルを配置
- 動画カバーの場合、最初の3秒でブランドと商品カテゴリが伝わること
- ライフスタイルシーンが商品単体写真より効果的

**商品画像:**
- 高解像度・均一な品質の商品画像を使用
- 白背景の統一感ある商品写真 or 統一テーマのライフスタイル写真
- 商品の選択: 自動（Metaが最適な商品を選択）or 手動選択

### 5-3. Instant Experienceテンプレートの選び方

| テンプレート | 概要 | 適したケース |
|-------------|------|-------------|
| **Instant Storefront（店頭）** | 商品をカタログ風に表示。最も推奨 | EC、複数商品の販売 |
| **Instant Lookbook** | ライフスタイル画像に商品タグ付け | アパレル、インテリア、コスメ |
| **Instant Customer Acquisition** | リード獲得やアクション促進に特化 | B2B、サービス、登録促進 |
| **Instant Storytelling** | 没入型のブランドストーリー | ブランド認知、新商品ローンチ |

### 5-4. コピーの書き方

- **見出し（25文字以内）:** カテゴリ名やオファーを端的に（例:「新作コレクション」「最大50%OFF」）
- **本文（90文字以内）:** ベネフィット＋行動喚起
- 極端に短い文字数制限のため、1つの訴求に絞り込む

## 6. リード広告（Lead Ads / Instant Forms）のクリエイティブ制作

### 6-1. 仕様

| 項目 | 仕様 |
|------|------|
| フォーマット | 画像 or 動画 + Instant Form |
| フォーム質問数 | 最大15問（推奨: 3問以内の追加質問） |
| フォームタイプ | 量重視（More Volume）/ 意図重視（Higher Intent） |
| 自動入力 | ユーザーのFacebook情報で氏名・メール・電話を自動入力 |

### 6-2. フォーム設計のベストプラクティス

**フォームタイプの選択:**

| タイプ | 特徴 | 適したケース |
|--------|------|-------------|
| **More Volume（量重視）** | 入力が簡単でフォーム完了率が高い | メルマガ登録、資料DL、初期リード獲得 |
| **Higher Intent（意図重視）** | 送信前に回答内容の確認ステップあり | デモ依頼、高額サービスの問い合わせ |
| **Rich Creative** | 画像、ブランディング、商品詳細を追加可能 | ブランド体験を重視する場合 |

**質問設計:**

```
[基本情報] 名前、メールアドレス（自動入力 → 摩擦最小）
    ↓
[セグメント質問] 1〜2問の選択式質問でリードを分類
  例: 「どのサービスに興味がありますか？」（複数選択）
  例: 「ご予算は？」（選択肢から選ぶ）
    ↓
[完了画面] サンクスメッセージ + 次のステップへのCTA
  例: 「ウェブサイトで詳細を見る」「担当者から24時間以内にご連絡」
```

**リード品質を高めるテクニック:**
- カスタム質問を追加する（意図の薄いリードをフィルタリング）
- SMS認証をONにする（偽の電話番号を排除）
- CRM連携を初日から設定し、コンバージョンデータをMetaにフィードバック → Conversion Leads最適化が可能に

### 6-3. 広告クリエイティブの設計

**画像/動画の設計:**
- フォームで何が得られるかを明確に伝える（「無料ガイドをダウンロード」「無料相談を予約」）
- オファーの価値を視覚的に示す（ガイドの表紙、セミナーの様子等）
- 画像広告で十分な場合が多い（動画は必須ではない）

**コピー:**
- **メインテキスト:** オファーの価値を簡潔に説明（何が得られるか、なぜ今か）
- **見出し:** オファー名を直接的に（「無料SEOガイド」「初回相談無料」）
- **CTA:** 「登録する」「今すぐダウンロード」「無料で試す」

### 6-4. Instant Form の「イントロ」セクション

フォーム表示前のイントロ画面で、以下を伝える:
- フォーム送信後に何が起きるか（「24時間以内にメールで資料をお届け」）
- ユーザーが得られるベネフィット（「〇〇が分かる無料ガイド」）
- 所要時間（「30秒で完了」）

## 7. Instant Experience（旧Canvas）のクリエイティブ制作

### 7-1. 概要

広告タップ時にフルスクリーンで展開されるインタラクティブなモバイル体験。コレクション広告のタップ先として使用されるほか、単独の広告フォーマットとしても利用可能。

### 7-2. デザインのベストプラクティス

**読み込み速度:**
- 通常のモバイルウェブページの15倍高速で読み込まれる
- この速度優位を活かし、LP代わりとしても有効

**推奨構成:**
```
[ヘッダー] ブランドロゴ + キャッチコピー
  ↓
[ヒーロー] 動画 or 画像でインパクト
  ↓
[ベネフィット] 3〜4つの特徴をビジュアルで紹介
  ↓
[社会的証明] テスティモニアル or 実績数値
  ↓
[商品/サービス] カルーセルまたはグリッドで紹介
  ↓
[CTA] 「今すぐ購入」「詳しく見る」等のボタン
```

## 8. クリエイティブテストと運用サイクル

### 8-1. テストの基本原則

- **1つのクリエイティブに対して3パターン以上のコピー**をテスト
- テスト期間は **約2週間** を目安に十分なデータを蓄積
- **1度に1変数のみ変更**してテスト（画像だけ変える、コピーだけ変える等）

### 8-2. 週次クリエイティブ運用サイクル

```
Week 1: 5〜10本の新しいクリエイティブを投入
  ↓
Week 2: 7日間、変更なしで配信（学習期間）
  ↓
Week 2末: 勝者/敗者を判定
  - 勝者 → スケーリングキャンペーンへ移動
  - 敗者 → 停止
  ↓
Week 3: パフォーマンスデータからインサイトを抽出
  ↓
Week 3: インサイトを元に次バッチのクリエイティブを制作
  ↓
繰り返し
```

### 8-3. 勝者の判断基準

| 指標 | 判断 |
|------|------|
| CTR（クリック率） | クリエイティブの訴求力。高いほど引きが強い |
| CPA / ROAS | ビジネス目標に対する効率。最終判断はこれ |
| フックレート（3秒視聴率） | 動画の冒頭の効果 |
| フリクエンシー | 3.0以上は疲労の兆候 |
| CPMr（1000リーチ単価） | $20以下が健全。上昇はオーディエンス疲労 |

### 8-4. クリエイティブの寿命管理

- **30〜60日** で勝ちクリエイティブもリフレッシュを検討
- フリクエンシー上昇 + CTR低下 = 疲労のサイン
- 疲労時の対応は **入札調整ではなくクリエイティブのリフレッシュ**
- 月に最低 **2〜4回** はクリエイティブを更新
- キャプションコピー、CTAデザイン、映像フレーミングの微調整で **CTR 10〜15%改善** が可能

### 8-5. スケーリング時のTips

- **Post IDの複製:** 勝ちクリエイティブを別キャンペーンにスケールする際、Post IDを使って複製するとソーシャルプルーフ（いいね・コメント・シェア）が引き継がれ、CTR/CVRが向上
- **コンセプト単位でスケール:** 個別の広告ではなく、勝った「コンセプト」（訴求軸×フォーマット）を異なるバリエーションで展開
- **予算増加は週20〜30%以内:** 急激な予算増加はAIの学習をリセットする

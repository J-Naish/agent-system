# Nano Banana Pro プロンプトガイド

画像生成プロンプトの詳細な書き方と用途別テンプレート集。

---

## 基本原則

### 1. 具体的かつ詳細に書く

曖昧な指示は曖昧な結果を生む。以下の要素を意識して含める:

- **被写体**: 何を描くか（人物、物、風景、抽象）
- **構図**: カメラアングル、距離感、フレーミング
- **スタイル**: 写真風、イラスト、水彩画、3Dレンダリング等
- **色調・雰囲気**: 暖色系、クール、ミニマル、レトロ等
- **テキスト**: 画像内に入れたい文字（ある場合）

悪い例:
```
おしゃれなカフェの画像
```

良い例:
```
自然光が差し込む北欧スタイルのカフェの店内写真。
白い壁と木製のテーブル。窓際の席にラテアートのコーヒーと
クロワッサンが置かれている。浅い被写界深度で背景はボケている。
温かみのある色調。
```

### 2. カメラ・写真用語を活用する

Nano Banana Proは写真用語への反応が特に良い:

| カテゴリ | 有効な用語例 |
|---------|-------------|
| アングル | ローアングル、ハイアングル、俯瞰、アイレベル、ダッチアングル |
| レンズ | 広角レンズ、望遠レンズ、マクロレンズ、魚眼レンズ、35mm、85mm |
| 被写界深度 | 浅い被写界深度、ボケ感、パンフォーカス、背景ぼかし |
| ライティング | 自然光、ゴールデンアワー、バックライト、リムライト、スタジオ照明 |
| 質感 | マット仕上げ、光沢、メタリック、テクスチャ、ガラス質 |

### 3. スタイル指定のパターン

```
# 写真風
「一眼レフで撮影した写真。自然光、浅い被写界深度」

# イラスト風
「フラットデザインのベクターイラスト。パステルカラー。線なし」

# 3Dレンダリング
「3Dレンダリングのアイソメトリックイラスト。柔らかい影。ミニチュア風」

# 水彩画風
「透明水彩で描かれた風景画。にじみと余白を活かした構図」

# マンガ・アニメ風
「日本のアニメスタイル。セル塗り。鮮やかな色彩」
```

## 写真風画像の生成

写真品質の画像を生成する際のテンプレート:

```
[被写体の詳細な説明]。
[カメラ設定: レンズ、アングル、被写界深度]。
[ライティング: 光源の種類と方向]。
[色調・雰囲気]。
[追加の指示: テキストなし、ウォーターマークなし等]
```

例:
```
東京・渋谷スクランブル交差点の夜景。雨上がりの路面に
ネオンが反射している。やや俯瞰のアングル。35mmレンズ。
長時間露光で通行人が流れるような軌跡になっている。
シアンとマゼンタの色調。映画のワンシーンのような雰囲気。
```

## 日本語テキスト入り画像

Nano Banana Proの最大の強みの一つが日本語テキストの高精度レンダリング。

### テキスト指示の書き方

1. **テキスト内容は「」で明確に囲む**
2. **フォントスタイルを指定する**（ゴシック体、明朝体、手書き風等）
3. **配置を指定する**（上部中央、左下、全面等）
4. **サイズ感を指示する**（大きく、小さく、画面の1/3等）

良い例:
```
画像の上部中央に「春の特別セール」と大きなゴシック体の白文字で配置。
文字には薄い影をつけて視認性を高める。
その下に小さく「3/1〜3/31 全品20%OFF」と添える。
```

悪い例:
```
セールの文字を入れて
```

### 日本語テキストの精度を上げるコツ

- 1つの画像に入れるテキスト量は少なめにする（多すぎると崩れやすい）
- 長い文章より短いキャッチコピーの方が精度が高い
- フォントの種類を指定すると安定する
- テキストの色と背景のコントラストを指示に含める
- 文字数が多い場合は「正確にこの文字を表示すること」と強調する

## ビジネス用途別テンプレート

### サムネイル / アイキャッチ

```
YouTubeサムネイル用の横長画像（16:9）。
左側に[人物/オブジェクトの説明]。
右側に「[タイトルテキスト]」と大きな太字のゴシック体。
文字色は白、文字の背後に半透明の黒いグラデーション。
背景は[色/パターン]のグラデーション。
インパクトがあり、小さなサイズでも読みやすいデザイン。
```

### 商品モックアップ

```
プロの商品撮影。白背景のスタジオ撮影。
[商品の詳細な説明]が画面中央に配置されている。
柔らかいスタジオ照明。商品の影は自然で柔らかい。
商品ラベルに「[ブランド名/商品名]」のテキスト。
高解像度、商品カタログに使えるクオリティ。
```

### インフォグラフィック / 図解

```
ビジネス向けのインフォグラフィック。白背景。
タイトル「[タイトル]」を上部に大きく配置。
[数字]つのポイントを縦に並べ、各ポイントに
アイコンと短い説明文を添える。
配色は[ブランドカラー]を基調とする。
フラットデザイン。清潔感のあるレイアウト。
```

### 不動産・物件広告

```
高級感のある不動産広告の画像。
[物件の外観/内観の説明]が大きく表示されている。
画像下部に半透明の帯を配置し、その上に
「[物件名]」と「[キャッチコピー]」を
白文字で表示。右下に「[会社名/ロゴ]」を小さく配置。
プロのカメラマンが撮影したような品質。
```

### SNS投稿用バナー

```

[商品/サービスの写真の説明]。

### 飲食店メニュー

```

料理の横に「[メニュー名]」と「[価格]」を

## 画像編集のプロンプト

画像編集（edit_image.sh）で使う場合の指示パターン:

### 背景の変更
```
この画像の被写体はそのままに、背景だけを[新しい背景の説明]に変更してください。
被写体のエッジは自然に馴染ませてください。
```

### テキストの追加
```
この画像の[位置: 上部/下部/中央]に
「[テキスト内容]」と[フォントスタイル]で追加してください。
文字色は[色]、[影/縁取り等の装飾]をつけてください。
元の画像の雰囲気は保持すること。
```

### スタイル変換
```
この写真を[スタイル: 水彩画風/アニメ風/ポップアート風]に変換してください。
被写体の特徴と構図は維持すること。
```

### 部分的な編集
```
この画像の[特定の部分]を[変更内容]に変更してください。
それ以外の部分は一切変更しないでください。
```

## よくある失敗と対策

### 日本語テキストが崩れる

対策:
- テキスト量を減らす（1画像につき2〜3フレーズまで）
- 「正確に」「一字一句間違えずに」と強調する
- フォントの種類を明示する
- Nano Banana Pro（gemini-3-pro-image-preview）の方が日本語精度が高い

### 意図と違う画像が生成される

対策:
- プロンプトの各要素を改行で区切り、構造化する
- 「〜ではなく」のネガティブ指示を活用する
- 参考画像がある場合は edit_image.sh で参照させる

### 人物がリアルすぎる / 不気味の谷

対策:
- 「イラスト風」「3Dレンダリング風」などスタイルを指定する
- 人物が不要な場合は「人物なし」と明記する

### 画像サイズ・アスペクト比が合わない

対策:
- プロンプトに明示する:「横長の16:9」「正方形の1:1」「縦長の9:16」
- 用途を書く:「YouTubeサムネイル用」「Instagramストーリー用」

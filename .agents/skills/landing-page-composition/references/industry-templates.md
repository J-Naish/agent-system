# 業種別LP構成テンプレート

業種ごとの鉄板セクション構成パターン。各テンプレートはそのまま使うのではなく、商材と状況に応じてカスタマイズすること。

---

## BtoB（SaaS、コンサル等）

```
1. ファーストビュー（キャッチコピー + デモCTA + 信頼シグナル）
2. 課題提起（業務上の痛み 3〜5個）
3. 解決策の提示（プロダクト概要 + 主要ベネフィット）
4. 機能紹介（3〜5機能、各機能をベネフィットに接続）
5. 導入事例（業種・課題・成果の3点セット × 2〜3社）
6. 導入企業ロゴ一覧
7. 料金プラン比較表
8. 利用の流れ（3〜4ステップ）
9. FAQ（5〜8問）
10. 最終CTA + 資料請求フォーム
```

**特徴:**
- 意思決定者が複数いるため、情報量は多めでOK
- 料金プランの比較表を入れると検討しやすい
- 「資料ダウンロード」をCTAにすると敷居が下がる
- 導入企業ロゴは信頼構築に非常に有効
- ROI（投資対効果）を数値で示すと意思決定を加速する

**中間CTAの配置候補:**
- 機能紹介の直後
- 導入事例の直後

---

## BtoC（EC、美容、健康等）

```
1. ファーストビュー（キャッチコピー + 限定オファー + ビジュアル）
2. 悩み共感（ターゲットの日常の悩み 3〜5個）
3. 商品紹介（解決策としての商品）
4. 成分・根拠（なぜ効果があるのかの裏付け）
5. ビフォーアフター / 使用者の変化
6. お客様の声（実名・写真付き × 3〜5件）
7. メディア掲載・受賞歴
8. 限定オファー（期間限定、数量限定、初回特典）
9. FAQ（5〜8問）
10. 最終CTA + 購入フォーム
```

**特徴:**
- 感情に訴える構成が有効
- ビフォーアフターや口コミの比重を大きくする
- 期間限定・数量限定の緊急性が効く
- ビジュアル（写真・動画）の品質が成約率に直結
- 権威性（専門家監修、メディア掲載）が信頼を補強

**中間CTAの配置候補:**
- 商品紹介の直後（衝動買い層向け）
- お客様の声の直後

---

## リフォーム・工務店

```
1. ファーストビュー（施工後ビジュアル + 無料見積もりCTA）
2. 施工事例（ビフォーアフター × 3〜5件）
3. 選ばれる理由（3〜5ポイント）
4. 工事の流れ（相談→現地調査→見積もり→施工→引き渡し）
5. お客様の声（写真付き × 3〜5件）
6. 対応エリア
7. FAQ（5〜8問、費用・工期・保証に関する質問）
8. 保証・アフターサービス
9. 最終CTA + 無料見積もりフォーム
```

**特徴:**
- ビジュアル（施工写真）が最重要
- 「地域密着」「アフター保証」が信頼シグナルになる
- 無料見積もりのハードルを下げるマイクロコピーが効く
- 工事の流れで「何が起きるか分からない」不安を解消
- 対応エリアの明示で無駄な問い合わせを防ぐ

**中間CTAの配置候補:**
- 施工事例の直後
- お客様の声の直後

---

## 教育・スクール・講座

```
1. ファーストビュー（受講後の成果 + 無料体験CTA）
2. こんな方におすすめ（ターゲット像 3〜5パターン）
3. カリキュラム概要（学べる内容のハイライト）
4. 選ばれる理由（他スクールとの差別化 3〜5ポイント）
5. 講師紹介（実績・経歴付き）
6. 受講生の声（成果付き × 3〜5件）
7. 受講の流れ（申込→受講→サポート→成果）
8. 料金プラン
9. FAQ（受講条件、難易度、サポート体制）
10. 最終CTA + 無料体験申込フォーム
```

**特徴:**
- 「受講後にどうなるか」の成果イメージが最重要
- 講師の権威性・実績が信頼を大きく左右する
- 無料体験・無料カウンセリングでハードルを下げる
- カリキュラムは詳細すぎず、成果に紐づけて紹介

**中間CTAの配置候補:**
- 受講生の声の直後
- 料金プランの直後

---

## 人材・採用

```
1. ファーストビュー（求める人材像 or 企業の魅力 + 応募CTA）
2. 会社紹介（ミッション・ビジョン・事業内容）
3. 仕事内容（具体的な業務と成長機会）
4. 社員インタビュー（日常の様子、やりがい × 2〜3名）
5. 福利厚生・制度（一覧表形式）
6. 選考の流れ（応募→書類選考→面接→内定）
7. FAQ（勤務条件、選考に関する質問）
8. 最終CTA + 応募フォーム
```

**特徴:**
- 「働くイメージ」が伝わる写真・動画が有効
- 社員の生の声が応募意欲を高める
- 選考の流れで「何をすればいいか」を明確にする
- 応募フォームは項目を絞る（詳細は選考過程で聞く）

---

## 士業・専門サービス（弁護士、税理士、コンサル等）

```
1. ファーストビュー（解決できる課題 + 無料相談CTA）
2. こんなお悩みありませんか（3〜5個）
3. サービス内容（対応領域の概要）
4. 選ばれる理由（専門性・実績・対応力 3〜5ポイント）
5. 代表・担当者紹介（経歴・資格・実績）
6. 解決事例（課題→対応→結果 × 2〜3件）
7. 料金体系（明瞭会計をアピール）
8. ご相談の流れ（3〜4ステップ）
9. FAQ（費用、秘密保持、対応範囲）
10. 最終CTA + 無料相談フォーム
```

**特徴:**
- 専門家の信頼性（資格、実績年数）が最重要
- 「初回無料相談」で心理的ハードルを下げる
- 料金の透明性が不安解消に直結
- 守秘義務・プライバシー保護の明示が必須

## セクション選択ガイド

すべてのセクションを入れる必要はない。以下を基準に取捨選択する:

### 必須セクション（全業種共通）
- ファーストビュー（キャッチコピー + CTA + ビジュアル）
- 課題共感 or ターゲット明示
- 解決策 / サービス紹介
- 社会的証明（お客様の声 or 導入事例）
- 最終CTA + フォーム

### 推奨セクション（あると効果的）
- 選ばれる理由（差別化）
- 利用の流れ
- FAQ
- 保証・安心材料

### 状況に応じて追加
- 料金プラン（価格が公開可能な場合）
- ビフォーアフター（成果が視覚的に示せる場合）
- メディア掲載・受賞歴（実績がある場合）
- 対応エリア（地域ビジネスの場合）
- 講師・担当者紹介（人が信頼の軸になる場合）

---

## セクション間の接続

セクションは独立するのではなく、物語として流れるように接続する:

1. **共感 → 解決**: 「その悩み、○○が解決します」
2. **解決 → 差別化**: 「なぜ○○が選ばれるのか」
3. **差別化 → 実績**: 「実際にこんな成果が出ています」
4. **実績 → 社会的証明**: 「お客様からもこんな声をいただいています」
5. **社会的証明 → CTA**: 「あなたも始めてみませんか？」

各セクション間の具体的な接続コピーは **copywriting** スキル（references/natural-transitions.md）を参照。
